# BB Bio Chapter 7 — Cardiovascular System

Scope: Anatomy of cardiovascular system (heart chambers, valves, conduction system, vessel types); blood (composition, RBCs, blood types, coagulation); physiology (cardiac cycle, output, BP regulation, capillary exchange).

AAMC FC mapping: FC3B (cardiovascular portion)

Kaplan: Biology Ch 7

---

HEART ANATOMY & BLOOD FLOW:
- Vena cava (SVC/IVC) -> RA -> tricuspid -> RV -> pulmonary valve -> pulmonary arteries -> lungs -> pulmonary veins -> LA -> mitral (bicuspid) -> LV -> aortic valve -> aorta
- Arteries AWAY from heart (pulmonary arteries carry deoxy); veins TOWARD heart (pulmonary veins carry oxy)
- LV thickest wall (highest pressure)
- Chordae tendineae + papillary muscles anchor AV valves (prevent prolapse)

CARDIAC CYCLE:
- Atrial systole (P wave; last ~20% ventricular fill)
- Isovolumetric contraction (AV valves close = S1 "lub")
- Ventricular ejection (QRS, semilunar valves open)
- Isovolumetric relaxation (semilunar valves close = S2 "dub"; dicrotic notch)
- Passive ventricular filling (~80% of fill)

CARDIAC OUTPUT:
- CO = HR x SV
- HR: SNS (NE on beta-1) up; PNS (vagal ACh muscarinic) down
- SV depends on:
  - Preload (end-diastolic stretch; Frank-Starling: more stretch -> stronger contraction up to a point)
  - Contractility (inotropy; SNS, Epi, digitalis increase)
  - Afterload (pressure vent must overcome; HTN raises -> SV down)

CONDUCTION SYSTEM (autorhythmic):
- SA node (60-100 bpm, pacemaker) -> AV node (~0.1 sec delay; PR interval) -> Bundle of His -> bundle branches -> Purkinje fibers (apex to base; QRS)
- Pacemaker: HCN "funny" channels (slow Na+ leak)

BP REGULATION:
- MAP = CO x TPR; ~ diastolic + 1/3(SBP - DBP)
- Short-term: baroreceptors (carotid sinus, aortic arch); high BP -> more firing -> more PNS, less SNS -> BP drops
- Vasovagal syncope: parasympathetic surge -> bradycardia + vasodilation -> faint
- Long-term: RAAS (renin -> ang I -> ACE -> ang II -> vasoconstrict + aldosterone + ADH + thirst)
- ANP: atrial stretch -> Na+/water excretion; opposes RAAS

VESSELS:
- Arteries: thick, elastic, high-pressure; recoil maintains diastolic flow
- Arterioles: thick smooth muscle; MAJOR site of TPR regulation
- Capillaries: single endothelial layer; slowest velocity (largest total cross-section)
- Venules; Veins: thin walls, valves, low pressure (capacitance vessels); skeletal muscle pump + respiratory pump assist return
- Starling forces: hydrostatic out, oncotic in; net filtration at arteriolar end, reabsorption at venular end; lymph collects excess

BLOOD COMPOSITION:
- Plasma (~55%): water, albumin, globulins, fibrinogen, electrolytes, gases, nutrients, wastes
- Formed elements (~45%): RBCs (99%), WBCs, platelets; hematocrit = % RBC
- RBCs: biconcave (max SA), no nucleus/mitochondria/organelles, anaerobic glycolysis only, ~120 day lifespan, EPO from kidneys, recycled in spleen (heme -> biliverdin -> bilirubin)

COAGULATION:
1. Vascular spasm
2. Platelet plug: vWF -> collagen binding, ADP/TXA2 release
3. Coagulation cascade: intrinsic + extrinsic -> common pathway -> Xa -> prothrombin to thrombin -> fibrinogen to fibrin; factor XIII cross-links
4. Fibrinolysis: plasmin from plasminogen

BLOOD TYPES:
- ABO: A antigen+anti-B; B antigen+anti-A; AB both antigens, no antibodies (universal recipient); O no antigens, both antibodies (universal donor); IA/IB codominant, both dominant over i
- Rh: Rh- mother + Rh+ fetus -> sensitization risks erythroblastosis fetalis in subsequent pregnancies; RhoGAM prevents
