MENDELIAN GENETICS:
- Law of segregation: 2 alleles per gene; gametes get 1 each (meiosis I)
- Law of independent assortment: genes on different chromosomes sort independently (random orientation at metaphase I; 2^23 combos in humans)
- Punnett squares; test cross (unknown x homozygous recessive)
- Monohybrid: Aa x Aa -> 1:2:1 genotype, 3:1 phenotype
- Dihybrid: AaBb x AaBb -> 9:3:3:1 phenotype (assumes independent assortment)
- 9:3:3:1 fails when: linked genes, epistasis, lethal alleles, non-Mendelian
- Dominant = usually functional protein; recessive = nonfunctional
- Dominant negative exception (mutant protein interferes; e.g., osteogenesis imperfecta collagen)

NON-MENDELIAN:
- Incomplete dominance: blended intermediate (RR x R'R' -> RR' pink); cross gives 1:2:1 phenotype
- Codominance: both expressed simultaneously (ABO -- IA IB = AB blood)
- Multiple alleles (ABO): 3 alleles; IA/IB codominant; both dominant over i; type O = universal donor; AB = universal recipient
- X-linked recessive: males more affected (hemizygous); no male-male transmission; carrier mothers; hemophilia, Duchenne, color blindness
- X-inactivation: random inactivation, Barr body; calico cats; Barr bodies = X count − 1
- Epistasis: one gene masks another (Lab coat: ee = yellow regardless of B); modified ratios 9:3:4, 12:3:1, 9:7
- Pleiotropy: 1 gene, many phenotypes (sickle cell, PKU)
- Polygenic: many genes -> 1 trait (continuous distribution: height, skin color)
- Penetrance (% with genotype showing phenotype) vs expressivity (severity among those affected)
- Genetic linkage: genes on same chromosome inherited together; recombination frequency = recombinants/total x 100; max 50%; 1% = 1 cM = 1 map unit
- Three-point cross: middle gene = least frequent double-crossover

MUTATIONS:

Point mutations:
- Silent: same AA (often wobble position; degenerate code)
- Missense: different AA (conservative or non-conservative; sickle cell GAG->GUG = Glu->Val)
- Nonsense: stop codon -> truncated
- Frameshift: insertion/deletion not multiple of 3 -> reading frame destroyed (most severe)

Chromosomal:
- Deletion (Cri-du-chat chr5)
- Duplication
- Inversion
- Translocation (Philadelphia chr t(9;22) BCR-ABL in CML; Burkitt t(8;14) c-Myc)
- Robertsonian (acrocentric fusion; familial Down syndrome via t(14;21))

Other patterns:
- Trinucleotide repeat expansion: Huntington (CAG, AD, chr 4), Fragile X (CGG, X-linked, FMR1)
- Anticipation: repeat number increases each generation -> earlier/severer onset
- Common diseases: sickle cell (missense), CF (deltaF508 deletion), Down (trisomy 21)

POPULATION GENETICS — HARDY-WEINBERG:
- Equations: p + q = 1; p^2 + 2pq + q^2 = 1
- 5 conditions: no selection, no mutation, no migration (gene flow), no drift (infinite pop), random mating
- Strategy: start with q^2 (homozygous recessive = visible phenotype)
- X-linked: affected males = q (hemizygous), affected females = q^2

NATURAL SELECTION:
- Directional: shifts mean to one extreme (antibiotic resistance, peppered moth)
- Stabilizing: favors intermediate, narrows variance (human birth weight)
- Disruptive: favors both extremes (finch beaks; can lead to speciation)
- Sexual selection: traits that improve mating success even if cost survival (peacock)
- Balancing selection: maintains multiple alleles
  - Heterozygote advantage (sickle cell trait + malaria resistance)
  - Frequency-dependent (rare phenotype advantage)

GENETIC DRIFT (random, non-adaptive, strongest in small pops):
- Bottleneck: dramatic pop reduction; non-representative survivors (elephant seals)
- Founder effect: small group establishes new pop (Amish/Ellis-van Creveld)

GENE FLOW (migration):
- Increases within-pop diversity; decreases between-pop differences
- Main force preventing speciation

SPECIATION:
- Allopatric: geographic separation (most common; Galapagos finches)
- Sympatric: no geographic barrier; polyploidy in plants, habitat/temporal isolation
- Reproductive isolation: prezygotic (temporal, behavioral, mechanical, gametic, habitat) vs postzygotic (hybrid inviable, sterile, breakdown — mules)

FITNESS & EVOLUTION:
- Fitness = reproductive success (NOT physical strength); relative to environment
- Inclusive fitness: kin selection
- Adaptive radiation: rapid diversification into new niches (post-extinction or new environment); Darwin's finches, mammals after K-Pg, cichlids
- Convergent evolution: unrelated -> similar traits (analogous structures: bird wing vs butterfly)
- Divergent evolution: related -> different traits (homologous structures: human arm vs whale flipper)

KEY FORMULAS:
- p + q = 1; p^2 + 2pq + q^2 = 1
- Recombination freq = recombinants / total × 100
- 1 cM = 1 map unit = 1% recombination
- Barr bodies = X count − 1

COMMON TRAPS:
- q vs q^2 (allele freq vs genotype freq of homozygous recessive)
- X-linked male = q (not q^2)
- Linked genes don't follow 9:3:3:1
- Incomplete dominance (blended) vs codominance (both expressed)
- Nondisjunction MI (all 4 abnormal) vs MII (2 normal + 2 abnormal)
- Penetrance (yes/no) vs expressivity (severity)
- Fitness ≠ strength
