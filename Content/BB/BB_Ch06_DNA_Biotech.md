# BB Chapter 6 — DNA & Biotechnology

Scope: DNA structure, eukaryotic chromosome organization, DNA replication, DNA repair, recombinant DNA / biotech techniques.

AAMC FC mapping: FC1B (Gene Expression)

Kaplan: Biochem Ch 6

---

DNA:
- Double helix, base pairing (A-T 2 H-bonds, G-C 3 H-bonds), antiparallel
- Major/minor grooves
- Eukaryotic chromosome organization: histones, nucleosomes, heterochromatin (condensed, silent) vs euchromatin (open, active)

REPLICATION:
- Semiconservative (Meselson-Stahl)
- Origin of replication, replication fork
- Helicase (unwinds), primase (RNA primer), DNA polymerase (5'->3' synthesis, 3'->5' proofreading), ligase (joins Okazaki fragments)
- Leading (continuous) vs lagging (discontinuous, Okazaki fragments)
- Telomeres/telomerase
- DNA repair: mismatch, nucleotide excision, base excision (Kaplan gap)

BIOTECHNOLOGY:
- PCR: denaturation (95C), annealing (55-65C), extension (72C), Taq polymerase, exponential amplification
- RT-PCR: mRNA -> cDNA via reverse transcriptase, then PCR
- qPCR: real-time quantification
- Blotting: Southern (DNA), Northern (RNA), Western (protein) -- SNoW DRoP
- Gel electrophoresis: smaller fragments migrate farther
- Restriction enzymes: palindromic recognition sites, sticky vs blunt ends
- DNA ligase, vectors/plasmids, cloning, transformation, selection markers
- Sanger sequencing (ddNTPs, chain termination)
- cDNA library (expressed genes, no introns) vs genomic library (complete DNA)
- Microarrays: genome-wide expression profiling
- CRISPR-Cas9: guide RNA-directed gene editing
- RFLP analysis
- ELISA (direct, indirect, sandwich) → see Ch 3
- FISH (fluorescence in situ hybridization) (Kaplan gap)

→ Transcription, translation, gene regulation: see Ch 7
