# BB — FC3B: Organ System Structure and Function (Bio)

Parent section: Foundational Concept 3 — Organ systems provide coordination

---

RESPIRATORY SYSTEM:
- Anatomy: nasal cavity -> pharynx -> larynx -> trachea -> bronchi -> bronchioles -> alveoli
- Ventilation: diaphragm contracts -> intrathoracic pressure drops -> air flows in (Boyle's law)
- Lung volumes: tidal, residual, vital capacity, total lung capacity
- Gas exchange: driven by partial pressure gradients (Dalton's law)
- O2-Hemoglobin dissociation curve:
  - Sigmoidal shape (cooperative binding)
  - RIGHT SHIFT = increased O2 unloading (Bohr effect): increased temp, increased CO2, decreased pH, increased 2,3-BPG
  - LEFT SHIFT = increased O2 affinity: opposite conditions, fetal Hb, CO poisoning
- CO2 transport: dissolved (7%), carbaminohemoglobin (23%), bicarbonate (70%)
  - Carbonic anhydrase: CO2 + H2O <-> H2CO3 <-> H+ + HCO3-

CARDIOVASCULAR SYSTEM:
- Heart: 4 chambers, valves (AV: tricuspid, mitral; semilunar: pulmonary, aortic)
- Cardiac cycle: systole (contraction) and diastole (relaxation)
- Cardiac output: CO = HR x SV
- Blood pressure: systolic/diastolic, regulated by baroreceptors, RAAS, ANP
- Vasovagal syncope: parasympathetic surge -> bradycardia + vasodilation -> BP drop -> fainting (baroreflex-related clinical)
- Vessel types: arteries (thick, elastic) -> arterioles (resistance vessels) -> capillaries (exchange) -> venules -> veins (valves, capacitance vessels)
- Blood: plasma (55%), RBCs (no nucleus, biconcave, 120-day lifespan), WBCs, platelets
- Coagulation cascade, fibrin clot
- Blood types: ABO (codominant IA, IB; recessive i) + Rh factor
- Fetal circulation: umbilical vein (oxygenated from placenta), ductus arteriosus (bypasses lungs), foramen ovale (R->L atrial shunt)

IMMUNE SYSTEM:
- Innate: physical barriers (skin, mucus, low pH, lysozyme), phagocytes (neutrophils -- first responders, macrophages, dendritic cells), complement system (MAC), interferons (antiviral), NK cells, inflammation (vasodilation, increased permeability, chemotaxis), fever
- Adaptive -- Humoral: B cells -> plasma cells (secrete antibodies) + memory B cells
  - Antibody structure: variable region (antigen binding), constant region (effector function)
  - Classes: IgG (most abundant, crosses placenta), IgM (first response, pentamer), IgA (mucosal), IgE (allergies, parasites), IgD (B cell surface)
  - Clonal selection, affinity maturation, class switching
- Adaptive -- Cell-mediated: T cells (mature in thymus)
  - CD4+ helper T cells: recognize MHC II (on APCs), activate B cells and macrophages
  - CD8+ cytotoxic T cells: recognize MHC I (on all nucleated cells), kill infected/cancer cells
  - MHC I: endogenous antigen presentation (all nucleated cells -> CD8)
  - MHC II: exogenous antigen presentation (APCs -> CD4)
- Active immunity (your own immune response, memory) vs passive (antibodies transferred, no memory)
- Vaccines: active immunity via weakened/inactivated pathogen
- Autoimmune diseases, immunodeficiency, hypersensitivity (Types I-IV)

DIGESTIVE SYSTEM:
- Mouth: salivary amylase (starch->maltose), lingual lipase
- Esophagus: peristalsis, LES
- ANS-GI: parasympathetic (vagus) increased motility/secretions; sympathetic (splanchnic) decreased -- blood shunted to muscle/heart
- Stomach: HCl (parietal cells), pepsinogen->pepsin (chief cells), mucus cells, gastrin (hormone)
- Small intestine: duodenum (receives bile + pancreatic juice), jejunum (main absorption), ileum (B12, bile salt reabsorption)
- Pancreas: trypsinogen->trypsin (activated by enterokinase), chymotrypsin, carboxypeptidase, pancreatic lipase, amylase, bicarbonate
- Liver: bile production (cholesterol derivative, emulsifies fat), detoxification, urea cycle, gluconeogenesis, plasma proteins, first-pass metabolism
- Gallbladder: stores and concentrates bile
- Large intestine: water absorption, bacterial flora (K, biotin production)
- Absorption: villi/microvilli (increased surface area), glucose/AAs via Na+ co-transport (secondary active), fats -> chylomicrons -> lymphatics

EXCRETORY/RENAL SYSTEM:
- Kidney: cortex, medulla, pelvis -> ureter -> bladder -> urethra
- Nephron: functional unit
  - Glomerulus/Bowman's capsule: filtration (GFR ~120 mL/min, filtered by size and charge)
  - PCT: bulk reabsorption (glucose, AAs, Na+, water, HCO3-), secretion (H+, drugs)
  - Loop of Henle: descending (water permeable, thin), ascending (Na+/K+/Cl- pump, impermeable to water)
  - DCT: fine-tuning (aldosterone -> Na+ reabsorb, PTH -> Ca2+ reabsorb)
  - Collecting duct: ADH -> aquaporin insertion -> water reabsorption
- Countercurrent multiplier: creates medullary osmotic gradient (300->1200 mOsm)
- RAAS: decreased BP -> renin -> angiotensin I -> ACE (lungs) -> angiotensin II -> vasoconstriction + aldosterone release -> Na+/H2O retention -> increased BP
- ANP: released by atria when stretched, opposes RAAS
- Acid-base balance: H+ secretion, HCO3- reabsorption
- Clearance: C = (U*V)/P

MUSCULOSKELETAL:
- Skeletal muscle: sarcomere structure (Z-line to Z-line), actin (thin filaments + tropomyosin + troponin), myosin (thick filaments, cross-bridges)
- Sliding filament theory: Ca2+ released from SR -> binds troponin -> tropomyosin moves -> myosin binds actin -> power stroke (ATP hydrolysis)
- Excitation-contraction coupling: AP -> T-tubules -> SR Ca2+ release
- Muscle fiber types: slow-twitch (type I, oxidative, endurance), fast-twitch (type II, glycolytic, power)
- Smooth muscle: involuntary, single-unit vs multi-unit, calmodulin-mediated
- Cardiac muscle: intercalated discs (gap junctions), autorhythmic, striated
- Bone: compact (osteons/Haversian systems) vs spongy (trabecular)
- Bone cells: osteoblasts (build), osteoclasts (resorb), osteocytes (maintain)
- Endochondral ossification (cartilage -> bone) vs intramembranous (direct)

REPRODUCTIVE AND DEVELOPMENT:
- Male: testes -> seminiferous tubules (spermatogenesis), Sertoli cells (support), Leydig cells (testosterone)
- Female: ovaries -> oogenesis, menstrual cycle (follicular phase -> ovulation -> luteal phase)
- Hormonal control: GnRH -> FSH/LH
- Embryology: fertilization -> cleavage -> morula -> blastula -> gastrulation -> neurulation -> organogenesis
- Germ layers: ectoderm (skin, nervous system), mesoderm (muscle, bone, blood, kidneys), endoderm (gut lining, lungs, liver, pancreas)

INTEGUMENTARY: skin layers (epidermis, dermis, hypodermis), thermoregulation
LYMPHATIC: lymph, lymph nodes, spleen (blood filter), thymus (T cell maturation)
