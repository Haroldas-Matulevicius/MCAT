# BB — FC2B: Prokaryotes and Viruses (Bio)

Parent section: Foundational Concept 2 — Cells and cellular assemblies

---

PROKARYOTES:
- Cell wall: peptidoglycan -- Gram+ (thick wall, crystal violet) vs Gram- (thin wall + outer membrane + LPS)
- Ribosomes: 70S (30S+50S)
- Flagella, pili (attachment, conjugation), plasmids
- Nucleoid region (no membrane-bound nucleus)
- Endospores, biofilms
- Binary fission
- Genetic transfer: transformation (uptake from environment), transduction (phage-mediated: generalized/specialized), conjugation (F plasmid, pilus)

VIRUSES:
- Structure: nucleic acid core (DNA or RNA, ss or ds) + capsid +/- envelope
- Lytic cycle: attachment -> injection -> replication -> assembly -> lysis
- Lysogenic cycle: integration as prophage -> latency -> can switch to lytic
- Retroviruses: RNA -> reverse transcriptase -> DNA -> integrase -> host genome
- Bacteriophages
- Prions: misfolded proteins (PrPSc), no nucleic acid (Kaplan gap)
- Fungi basics: chitin cell wall, ergosterol membrane (Kaplan gap)
