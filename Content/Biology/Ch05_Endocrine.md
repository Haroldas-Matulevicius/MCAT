# BB Bio Chapter 5 — Endocrine System

Scope: Mechanisms of hormone action (peptide vs steroid, second messengers, intracellular receptors); endocrine organs and hormones (hypothalamus, pituitary, thyroid, parathyroid, adrenal, pancreas, gonads, pineal); feedback regulation.

AAMC FC mapping: FC3A (Endocrine system)

Kaplan: Biology Ch 5

---

HORMONE TYPES:

Peptide / protein / catecholamines (Epi):
- Hydrophilic, dissolve in blood, CANNOT cross membrane
- Cell-surface receptors (GPCRs, RTKs)
- Second messengers (cAMP, IP3/DAG, Ca2+)
- Fast (sec-min); short half-life
- Examples: insulin, glucagon, GH, ADH, oxytocin, Epi, all anterior pituitary

Steroid + thyroid:
- Lipophilic; carrier proteins in blood (TBG, SHBG); cross membrane
- Intracellular (cytoplasmic/nuclear) receptors -> transcription factor -> gene expression
- Slow (hours-days); long half-life
- Examples: cortisol, aldosterone, testosterone, estrogen, progesterone, T3/T4, calcitriol
- TRAP: T3/T4 are AA derivatives but behave like steroids; Epi is AA derivative but behaves like peptide

HYPOTHALAMIC-PITUITARY AXIS:
- Hypothalamus: neural -> hormonal converter
- Anterior pituitary (adenohypophysis): connected by hypothalamic-hypophyseal portal system; receives releasing/inhibiting hormones
- Posterior pituitary (neurohypophysis): direct neural extension; ADH and oxytocin made in hypothalamic nuclei (supraoptic, paraventricular), released from axon terminals

ANTERIOR PITUITARY HORMONES (FLAT PIG):
- GH (somatotropin): GHRH stim, somatostatin inhib; -> liver IGF-1; growth, raises glucose, protein synthesis
- TSH: TRH stim; -> thyroid -> T3/T4
- ACTH: CRH stim; -> adrenal cortex -> cortisol
- FSH: GnRH stim; F: follicle development/estrogen; M: spermatogenesis (Sertoli)
- LH: GnRH stim; F: ovulation, corpus luteum, progesterone; M: testosterone (Leydig)
- Prolactin: dopamine INHIBITS (tonic), TRH stim; mammary -> milk
- Stalk severed: all anterior pituitary hormones DECREASE except prolactin which INCREASES

POSTERIOR PITUITARY HORMONES:
- ADH (vasopressin): high osmolarity / low BV -> supraoptic nucleus; collecting duct aquaporin-2 insertion -> water reabs; vasoconstriction at high doses; DI = low ADH; SIADH = excess; alcohol inhibits
- Oxytocin: cervical stretch / nipple stim -> paraventricular nucleus; uterine contraction (positive feedback during labor), milk letdown

THYROID:
- T3/T4: from tyrosine + iodine; T4 -> T3 peripherally (T3 = active); raise BMR; nuclear receptor -> gene expression; required for brain development (cretinism if congenital deficiency)
- Regulation: TRH -> TSH -> T3/T4 -> negative feedback
- Graves: autoimmune TSH-mimic, hyperthyroidism, low TSH, exophthalmos
- Hashimoto: autoimmune destruction, hypothyroidism, high TSH
- Calcitonin (parafollicular C cells): LOWERS Ca2+ (inhibits osteoclasts); "calciTONin TONes down"

PARATHYROID:
- PTH: released when blood Ca2+ LOW; RAISES Ca2+ via:
  1. Bone resorption (osteoclasts via RANKL)
  2. Kidney: increases Ca2+ reabs (DCT), decreases phosphate reabs
  3. Activates vitamin D (1-alpha-hydroxylase -> calcitriol -> intestinal Ca2+ absorption)
- "PTH Brings Kalcium In" (Bone, Kidney, Intestine)

ADRENAL CORTEX (GFR — salt, sugar, sex):
- Zona Glomerulosa: aldosterone (mineralocorticoid; Na+ reabs/K+ secr in collecting duct; regulated by RAAS and high K+)
- Zona Fasciculata: cortisol (glucocorticoid; raises glucose -- gluconeogenesis, proteolysis, lipolysis; anti-inflammatory; CRH -> ACTH; diurnal rhythm)
- Zona Reticularis: DHEA (weak androgen)

ADRENAL MEDULLA: Epi (~80%) + NE (~20%) into blood; modified sympathetic postganglionic; ACh from preganglionic -> chromaffin cells

RAAS:
- Low BP -> JG cells -> renin -> angiotensinogen (liver) -> ang I -> ACE (lungs) -> ang II
- Ang II: vasoconstrict, aldosterone release, ADH release, thirst
- ACE inhibitors block this

PANCREAS (islets of Langerhans):
- Beta cells -> insulin (high glucose, fed): GLUT4 insertion (muscle/adipose), glycogenesis, lipogenesis, protein synthesis; LOWERS glucose (only one)
- Alpha cells -> glucagon (low glucose, fasted): glycogenolysis, gluconeogenesis, lipolysis, ketogenesis (liver)
- Delta cells -> somatostatin (paracrine, inhibits both)
- Type 1 DM: autoimmune beta cell destruction; Type 2: insulin resistance

GONADS:
- Testes: testosterone (Leydig, by LH); Sertoli (FSH; spermatogenesis support, inhibin neg-feedback FSH)
- Ovaries: estrogen (granulosa, FSH), progesterone (corpus luteum, LH)

PINEAL: melatonin (from serotonin); SCN -> circadian rhythm; rises in darkness

FEEDBACK:
- Negative (most hormones): T3/T4 -> -TRH/TSH; cortisol -> -CRH/ACTH
- Positive (rare): oxytocin in labor; LH surge at ovulation; clotting cascade
- Exogenous hormone X -> upstream drops (e.g., exogenous testosterone -> LH/FSH drop; basis of male contraception)

NERVOUS-ENDOCRINE INTEGRATION:
- Hypothalamus = bridge
- Sympathetic -> adrenal medulla
- Stress: SNS (immediate) + HPA (sustained cortisol)
- Blood glucose: SNS + Epi + cortisol all raise; insulin lowers
- Circadian: SCN -> melatonin (pineal) + cortisol rhythm
- ADH and BP: baroreceptors -> hypothalamus -> ADH
