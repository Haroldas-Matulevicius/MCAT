# BB Bio Chapter 8 — Immune System

Scope: Structure of the immune system; innate immune system (barriers, cellular components, complement, inflammation, interferons); adaptive immune system (B cells/humoral, T cells/cell-mediated, MHC I vs II, antibody classes); lymphatic system.

AAMC FC mapping: FC3B (immune portion)

Kaplan: Biology Ch 8

---

INNATE IMMUNITY (immediate, nonspecific, no memory):

Barriers:
- Skin, mucous membranes, stomach acid (pH ~2), lysozyme (tears, saliva), defensins, normal flora

Cellular:
- Neutrophils: most abundant WBC, FIRST RESPONDERS, phagocytes, ROS, NETs, short-lived
- Macrophages: from monocytes; phagocytose, APC (MHC II), cytokines (TNF-alpha, IL-1, IL-6)
- Dendritic cells: most effective APCs; bridge innate-adaptive
- NK cells: kill via missing-self (no MHC I); perforin + granzymes
- Mast cells: histamine -> vasodil, increased perm; allergic responses
- Eosinophils: parasites, allergy
- Basophils: histamine, heparin

Complement system:
- Classical, lectin, alternative pathways converge on C3 -> C3a + C3b
- C3b: opsonization
- C3a, C5a: anaphylatoxins
- C5b-C9: MAC -> lysis

Inflammation cascade:
- Damage -> mast cells release histamine -> vasodil + permeability up -> edema
- Neutrophils first, then macrophages
- Cardinal signs: rubor, calor, tumor, dolor, loss of function

Interferons:
- Released by virus-infected cells; alert neighbors (don't kill virus directly)
- Type I (IFN-alpha, beta): antiviral
- Type II (IFN-gamma): activates macrophages, NK

Fever: pyrogens (IL-1, IL-6, TNF) -> hypothalamus raises set point

ADAPTIVE — HUMORAL (B cell-mediated, extracellular):
- B cell -> matches antigen -> internalizes -> presents on MHC II to CD4+ T_H2 -> activated -> clonal expansion
  - Plasma cells (antibody factories, short-lived)
  - Memory B cells (long-lived, secondary response)
- Antibody structure: 2 heavy + 2 light chains, disulfide; variable region (Fab, antigen binding) + constant region (Fc, effector)
- Classes:
  - IgG: most abundant, monomer, ONLY class crossing placenta; opsonization, complement, neutralization
  - IgM: pentamer (J chain), first in primary response; agglutination, complement; blood typing
  - IgA: dimer in secretions; mucosal immunity, breast milk
  - IgE: mast cells/basophils; allergies, parasites
  - IgD: surface of mature naive B cells

ADAPTIVE — CELL-MEDIATED (T cell-mediated, intracellular):
- T cells: from bone marrow, mature in thymus (positive + negative selection)
- CD4+ helper T cells: recognize MHC II on APCs; release cytokines
  - T_H1 -> activate macrophages, CD8+
  - T_H2 -> activate B cells
  - T_H17 -> recruit neutrophils
- CD8+ cytotoxic T cells: recognize MHC I (all nucleated cells); perforin + granzymes -> apoptosis

MHC I vs II:
- MHC I: ALL nucleated cells (not RBCs); endogenous antigens (made inside); CD8+
- MHC II: APCs only (macrophages, dendritic, B); exogenous antigens; CD4+

ACTIVE vs PASSIVE IMMUNITY:
- Active: own response, memory; natural (infection) or artificial (vaccine); slow, long-lasting
- Passive: preformed antibodies, no memory; natural (maternal IgG, IgA in breast milk) or artificial (antiserum/antitoxin); immediate, short-lived

HYPERSENSITIVITY:
- Type I (immediate/anaphylactic): IgE on mast cells -> histamine; allergies, anaphylaxis
- Type II (cytotoxic): IgG/IgM on cell surface -> complement/NK destruction; transfusion reactions, HDN
- Type III (immune complex): Ag-Ab complex deposition -> complement; serum sickness, lupus
- Type IV (delayed): T cell mediated, no antibodies; 24-72 hr; PPD, contact dermatitis, transplant rejection
- Types I-III antibody mediated; IV cell mediated

LYMPHATIC SYSTEM:
- Functions: fluid return (-> thoracic duct -> L subclavian), fat absorption (lacteals -> chylomicrons), immune surveillance
- Lymph nodes: filter lymph; B cells in follicles, T cells in paracortex
- Spleen: filters BLOOD; red pulp (RBC destruction, reservoir), white pulp (lymphoid)
- Thymus: T cell maturation; involutes with age
- MALT: tonsils, Peyer's patches, appendix
- No pump — moved by skeletal muscle, breathing, smooth muscle; one-way valves
