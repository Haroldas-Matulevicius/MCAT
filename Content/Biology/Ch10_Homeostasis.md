# BB Bio Chapter 10 — Homeostasis (Excretory & Skin)

Scope: Excretory system (kidney/nephron anatomy, filtration/reabsorption/secretion, countercurrent multiplier, RAAS, ADH, acid-base, clearance); skin (epidermis layers, dermis, thermoregulation, vitamin D synthesis).

AAMC FC mapping: FC3B (excretory + integumentary portions)

Kaplan: Biology Ch 10

---

NEPHRON STRUCTURE (functional unit; ~1 million per kidney):

1. Glomerulus: tuft of fenestrated capillaries in Bowman's capsule; afferent in, efferent out; high hydrostatic P forces filtrate
   - Filtration barrier: fenestrated endothelium + basement membrane + podocyte slits; passes water, ions, glucose, AAs, urea; blocks proteins/cells

2. Bowman's capsule: collects filtrate (~protein-free plasma)

3. Proximal convoluted tubule (PCT): bulk reabsorption (~65-70%)
   - Reabsorbs: glucose (100% via SGLT), AAs, Na+, HCO3-, water (obligatory), urea, phosphate, K+
   - Secretes: H+, organic acids, drugs, toxins
   - Brush border + abundant mitochondria; Na+/K+ ATPase basolateral drives reabs

4. Loop of Henle: creates medullary gradient
   - Descending: water-permeable, solute-impermeable; water leaves -> filtrate concentrated
   - Thin ascending: water-impermeable, NaCl passively out
   - Thick ascending: water-impermeable; Na+/K+/2Cl- cotransporter (loop diuretics target); dilutes filtrate, concentrates medulla

5. Distal convoluted tubule (DCT): fine-tuning
   - Aldosterone -> Na+ reabs / K+ secretion
   - PTH -> Ca2+ reabs
   - Macula densa (JGA) senses NaCl -> renin

6. Collecting duct: final concentration
   - ADH -> aquaporin-2 insertion -> water reabs (concentrated urine); without ADH = dilute urine (DI)
   - Aldosterone (Na+ in / K+ out)
   - Intercalated cells: H+ secretion (acid-base)

THREE RENAL PROCESSES:
- Filtration (glomerulus): GFR ~125 mL/min (~180 L/day; we reabsorb ~99%)
- Reabsorption (PCT mostly; loop, DCT, CD): tubule -> peritubular capillaries
- Secretion (PCT, DCT, CD): peritubular caps -> tubule
- Excreted = Filtered - Reabsorbed + Secreted

COUNTERCURRENT MULTIPLIER:
- Thick ascending pumps NaCl out -> medulla salty
- Descending: water leaves into hypertonic medulla -> filtrate concentrates
- Hairpin "multiplies" gradient (300 cortex -> 1200 inner medulla mOsm)
- Vasa recta: countercurrent exchanger; preserves gradient
- Urea recycling (ADH increases urea permeability in inner medullary CD): ~50% of medullary osmolality

RAAS (renal perspective):
- Decreased perfusion / decreased NaCl at macula densa / sympathetic -> JG cells -> renin
- Renin: angiotensinogen -> ang I; ACE (lungs) -> ang II
- Ang II: efferent constriction (preserves GFR), systemic vasoconstrict, aldosterone, ADH, thirst, PCT Na+ reabs

ADH:
- Stimulus: high plasma osmolarity (osmoreceptors) or low BV/BP (baroreceptors)
- Released from posterior pituitary; V2 receptors on CD -> cAMP/PKA -> aquaporin-2 insertion
- Alcohol inhibits -> dilute urine -> dehydration

ACID-BASE:
- Acidosis: more H+ secretion, more HCO3- reabs, generate new HCO3- from glutamine -> NH4+ excreted
- Alkalosis: less H+ secretion, less HCO3- reabs

CLEARANCE:
- C = (U x V) / P
- Inulin (or creatinine clinically): freely filtered, not reabs/secr -> clearance = GFR
- C > GFR: filtered + secreted (PAH at low conc)
- C < GFR: filtered + reabsorbed (glucose, AAs, Na+)
- C = 0: completely reabsorbed (glucose at normal levels)
- PAH clearance ≈ renal plasma flow (~625 mL/min)
- Filtration fraction = GFR/RPF ~20%

INTEGUMENTARY SYSTEM (SKIN):

Layers (superficial to deep):
- Epidermis (avascular, stratified squamous):
  - Stratum basale (stem cells, melanocytes, Merkel cells)
  - Stratum spinosum (Langerhans dendritic/immune cells)
  - Stratum granulosum (keratohyalin, lipid secretion)
  - Stratum lucidum (only thick skin: palms, soles)
  - Stratum corneum (dead keratinized cells; main barrier)
- Dermis (CT with vessels, nerves, follicles, glands; collagen, elastin):
  - Papillary (loose CT, fingerprints)
  - Reticular (dense irregular CT)
- Hypodermis (subcutaneous, adipose; insulation, energy)

THERMOREGULATION:
- Heat loss: vasodilation, sweating (evaporative cooling)
- Heat conservation: vasoconstriction, piloerection, shivering
- Hypothalamus = thermostat

OTHER SKIN FUNCTIONS:
- Physical barrier; UV protection (melanin)
- Vitamin D synthesis (UV-B converts 7-dehydrocholesterol -> cholecalciferol)
- Sensation: mechano-, thermo-, nociceptors
- Immune: Langerhans cells
