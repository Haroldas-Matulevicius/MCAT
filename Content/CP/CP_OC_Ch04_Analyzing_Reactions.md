KEY pKas (memorize):
- HCl, HBr, HI, H2SO4: -7 to -10 (strong)
- R-COOH: ~4-5 (acetic 4.76)
- Phenol Ar-OH: ~10 (resonance)
- Ammonium R-NH3+: ~10
- Water: 15.7
- R-OH: ~16-18
- α-H to carbonyl: ~20 ketone, ~25 ester
- Terminal alkyne R-C≡C-H: ~25
- Amine R2N-H: ~36-38
- Alkane C-H: ~50

ACID-BASE EQUILIBRIUM:
- Reaction works if base is ≥4 pKa units > acid's conjugate
- Equilibrium favors weaker acid + weaker base (higher pKa side)

ACIDITY RATIONALE:
- Resonance: carboxylate (2 O's) > alkoxide (1 O)
- Electronegativity: O-H > N-H > C-H
- Hybridization: sp > sp² > sp³ C-H
- Inductive: EWGs (Cl, F, NO2) lower pKa (CCl3-COOH 0.7 << CH3-COOH 4.76)

NUCLEOPHILES (electron-rich, donate e- pair, Lewis base):
- Strength: charged > neutral; less EN better in protic; reverses in polar aprotic
- Bulky → poor nuc but good base (t-BuO-, LDA)
- Common: OH-, RO-, RS-, CN-, NH3, RNH2, H2O, halides, organometallics

ELECTROPHILES (electron-poor, accept e- pair, Lewis acid):
- Carbonyl C (δ+); carbocations; alkyl halide C; halogens (Cl2, Br2); H+

LEAVING GROUPS:
- Quality ∝ stability as anion ↔ inversely w/ conjugate acid pKa
- Excellent: I-, Br-, Cl-, OTs, OMs
- Good: H2O (from ROH2+) — protonate alcohols first
- Poor: F-, OH-, NH2-, OR- (strong bases, stay attached)

OXIDATION/REDUCTION IN ORGO:
- Oxidation: ↑bonds to heteroatom (O, N, X) or ↓bonds to H
- Reduction: opposite
- C oxidation levels (by # bonds to heteroatom):
  - 0: alkane
  - 1: alcohol, amine, alkyl halide, alkene
  - 2: aldehyde, ketone, gem-diol, hemiacetal
  - 3: carb acid, ester, amide, anhydride, acyl halide, nitrile
  - 4: CO2

OXIDIZING AGENTS:
- KMnO4 (strong; full ox of alcohols; cleaves alkenes)
- CrO3/H2SO4 (Jones), Na2Cr2O7 — primary OH → carb acid
- PCC (mild) — primary OH → aldehyde (stops there)
- Tollens (Ag(NH3)2+) — aldehyde → carb acid (silver mirror)
- O3 + Zn/Me2S — cleaves alkenes to carbonyls

REDUCING AGENTS:
- LiAlH4 (LAH): strong; reduces ALL carbonyls (aldehyde, ketone, ester, carb acid, amide); anhydrous
- NaBH4: mild; only aldehyde + ketone → alcohol
- H2/Pd or Pt or Ni: hydrogenation of alkenes/alkynes; sometimes carbonyls

3° ALCOHOLS RESIST OXIDATION:
- No H on C-OH carbon to remove

SELECTIVITY:
- Chemo: which FG reacts (NaBH4 picks ketone over ester)
- Regio: where on molecule (Markovnikov)
- Stereo: which stereoisomer (SN2 inversion)
- Stereospecific: mechanism REQUIRES (not just prefers)

SIX-STEP PROTOCOL:
1. ID functional groups (substrate + reagent)
2. Classify reagent (nuc/electroph/ox/red/acid/base)
3. ID most reactive site
4. Predict mechanism type
5. Apply pKa / stability reasoning
6. Check stereo + regio chem

QUICK PATTERNS:
- Strong base + acidic H → deprotonate (check pKa)
- Strong nuc + alkyl halide → SN2 (or E2 if bulky)
- Carbonyl + nuc → nucleophilic addition (aldehyde/ketone) or acyl substitution (derivative)
- Carbonyl + α-H + base → enolate/aldol
- Alkene + HX/X2/H2O+H+ → electrophilic addition, Markovnikov
- Alcohol + oxidizer → ox per type/strength
- Carbonyl + LAH/NaBH4 → alcohol
- Acid + alcohol + heat → Fischer esterification
- Ester + base + H2O → saponification (irreversible)
- Amide + acid/base + heat → hydrolysis
