ALCOHOL PROPERTIES:
- 1°/2°/3° based on # adjacent C
- H-bonding → high BP, water solubility (small alcohols)
- pKa: methanol 15.5, primary > secondary > tertiary (alkyl groups destabilize alkoxide)
- Phenol pKa ~10 (resonance-stabilized phenolate)

SN1/SN2/E1/E2 DECISION:

| | SN2 | SN1 | E2 | E1 |
|---|---|---|---|---|
| Steps | 1 (concerted) | 2 (carbocation) | 1 (concerted) | 2 (carbocation) |
| Rate | k[sub][nuc] | k[sub] | k[sub][base] | k[sub] |
| Stereo | Inversion | Racemization | Anti-periplanar | None |
| Best substrate | Methyl > 1° > 2° | 3° > 2° | 3° > 2° > 1° | 3° > 2° |
| Rearrangements | No | YES | No | YES |

DECISION TREE:
- Methyl/1°: strong nuc → SN2; bulky base → E2; SN1/E1 don't happen (no 1° carbocation)
- 2° (tricky): strong nuc → SN2 (some E2); bulky base → E2; weak/protic → SN1/E1 mix; heat → elimination
- 3°: strong base → E2; weak/protic → SN1/E1 mix; SN2 IMPOSSIBLE (steric)

VARIABLES:
- Nucleophile strength: charged > neutral; CN-, RS-, I-, OH-, RO-, NH3, N3- (strong); H2O, ROH (weak)
- Base strength: strong (OH-, RO-, NaH, LDA, tBuOK); bulky → E2 only
- Solvent: polar protic (H2O, alcohols) → SN1/E1; polar aprotic (DMSO, DMF, acetone) → SN2
- Heat → elimination

ALCOHOL → HALIDE (activate -OH first):
- HX (HCl, HBr, HI): 3° SN1 fast; 1°/2° SN2 slower (HI best)
- PBr3 → R-Br (1°/2°, SN2, inversion)
- SOCl2 → R-Cl (1°/2°, inversion via chlorosulfite)
- TsCl/MsCl + pyridine → R-OTs/R-OMs (excellent LG for downstream SN2)

DEHYDRATION (alcohol → alkene):
- H2SO4 or H3PO4, heat
- 2°/3°: E1 (carbocation; can rearrange)
- 1°: E2 (concerted)
- Zaitsev: more substituted alkene = major product
- POCl3/pyridine: mild, no rearrangement

OXIDATION OF ALCOHOLS:
- 1° + PCC → aldehyde (stops there)
- 1° + KMnO4 / CrO3-H2SO4 (Jones) / Na2Cr2O7 → carboxylic acid (full)
- 2° + any → ketone (resists further ox)
- 3°: NO OXIDATION (no H on -OH carbon)

REDUCTION:
- LiAlH4 (strong, anhydrous): all carbonyls → alcohol/amine
- NaBH4 (mild): aldehyde/ketone → alcohol; doesn't touch ester/carb acid/amide

PHENOLS:
- pKa ~10, resonance-stabilized phenolate
- Deprotonated by NaOH (not NaHCO3)
- Activate ring toward EAS at ortho/para (OH = strong o/p director)
- Easy oxidation → quinone (biologically important: CoQ, vitamin K, E)
- Williamson ether: phenolate + R-X → Ar-O-R
