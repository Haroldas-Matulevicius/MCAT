CARBONYL PROPERTIES:
- Aldehyde R-CHO (terminal); ketone R-CO-R' (internal)
- C=O: C is δ+ (electrophilic), O is δ- and H-bond acceptor
- Polar but no H-bond donor → moderate BP, partial water solubility
- IR: sharp strong C=O at 1700-1750 cm⁻¹ (most diagnostic peak)

REACTIVITY (aldehyde > ketone):
- Steric: 2 R groups in ketone vs 1 R + 1 H in aldehyde
- Electronic: alkyl groups donate e- (inductive), stabilize δ+ on ketone C → less electrophilic

NUCLEOPHILIC ADDITION MECHANISM:
1. Nuc attacks carbonyl C
2. π bond breaks, e- to O (alkoxide)
3. Tetrahedral intermediate
4. O picks up H+ (or stays alkoxide anhydrous)
- No leaving group → tetrahedral intermediate stays (vs acyl substitution in derivatives)
- Acid catalysis: protonate O first → more electrophilic; activates weak nucs
- Base catalysis: deprotonate nuc first → stronger nuc

KEY ADDITION REACTIONS:

| Nuc | Product | Conditions |
|---|---|---|
| ROH (1 eq) | Hemiacetal/hemiketal | Acid |
| ROH (2 eq) | Acetal/ketal | Acid, -H2O |
| RNH2 (1° amine) | Imine (Schiff base, C=N) | Mild acid, -H2O |
| R2NH (2° amine) | Enamine (C=C-N) | Mild acid, -H2O |
| HCN | Cyanohydrin | Base |
| RMgBr (Grignard) | Alcohol | Anhydrous, then H3O+ |
| NaBH4 / LAH | Alcohol (reduction) | NaBH4 mild, LAH strong |
| H2O | Hydrate (gem-diol) | Eq usually unfavored (formaldehyde, chloral exceptions) |

ACETAL FORMATION:
- Aldehyde + 1 ROH + acid → hemiacetal R-CH(OH)(OR')
- Hemiacetal + ROH + acid + remove H2O → acetal R-CH(OR')2
- Same for ketones (hemiketal → ketal)
- IS sugar cyclization! Glucose C1 aldehyde + C5 -OH (intramolecular) → pyranose ring; anomeric C → α/β anomers
- Acetals = PROTECTING GROUPS (stable in base, cleave in acid)

IMINE/ENAMINE:
- 1° amine + carbonyl → imine (C=N), reversible
- 2° amine + carbonyl → enamine (C=C-N), reversible
- MCAT connection: PLP (pyridoxal-5-P) uses Schiff base in transamination

CYANOHYDRIN:
- R-CHO + HCN → R-CH(OH)(CN)
- Add 1 carbon to chain; nitrile hydrolyzes to carb acid

GRIGNARD ADDITION:
- R-MgX from R-X + Mg in dry ether
- Strong nucleophilic carbanion equivalent
- Anhydrous! Water destroys Grignard (R-MgX + H2O → R-H)
- Formaldehyde (HCHO) + R-MgX → 1° alcohol
- Aldehyde + R-MgX → 2° alcohol
- Ketone + R-MgX → 3° alcohol
- Ester + 2 R-MgX → 3° alcohol w/ 2 identical R
- CO2 + R-MgX → carboxylic acid (after workup)

REDUCTION (carbonyl → alcohol):
- NaBH4: aldehyde→1° OH; ketone→2° OH; doesn't touch ester/carb acid
- LAH: aldehyde, ketone, ester, carb acid, amide (anhydrous)
- H2/metal catalyst: slower than alkene hydrogenation

OXIDATION:
- Aldehyde → carb acid (easy)
  - Tollens (Ag(NH3)2+): silver mirror = positive for aldehydes + reducing sugars
  - KMnO4, CrO3, Jones, K2Cr2O7
  - Benedict/Fehling (Cu2+): brick-red Cu2O ppt = positive for aldehydes + reducing sugars (urine glucose test)
- Ketone: NO easy oxidation (no H on carbonyl C)
