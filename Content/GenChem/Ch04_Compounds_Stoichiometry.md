# CP Gen Chem Chapter 4 — Compounds & Stoichiometry

Scope: MW, moles, formulas, balancing, reaction types, limiting reagent, yield, oxidation numbers, equivalents/normality.

AAMC FC mapping: FC4E (stoichiometry)

Kaplan: General Chemistry Ch 4

---

MOLE CONCEPT:
- 1 mol = 6.022 × 10^23 particles (Avogadro)
- MW (g/mol) numerically = atomic mass in amu
- STP: 22.4 L/mol (0°C, 1 atm)

FORMULAS:
- Empirical: simplest whole-number ratio (CH2O)
- Molecular: actual atoms (C6H12O6)
- Empirical from % comp: assume 100 g, → mol, divide by smallest, round
- Molecular = empirical × (actual MW / empirical MW)

BALANCING EQUATIONS:
1. Metals/unique elements first
2. Other nonmetals (not H, O)
3. H
4. O last
- Combustion: C → H → O

REACTION TYPES:
- Combination: A + B → AB (sometimes redox)
- Decomposition: AB → A + B (sometimes redox)
- Single displacement: A + BC → AC + B (ALWAYS redox); activity series (K>Na>Ca>Mg>Al>Zn>Fe>Pb>H>Cu>Ag>Au)
- Double displacement: AB + CD → AD + CB (rarely redox); precipitation, neutralization
- Combustion: CxHy + O2 → CO2 + H2O (ALWAYS redox + exothermic); incomplete → CO or soot
- Disproportionation: same element both oxidized and reduced (2 H2O2 → 2 H2O + O2)

LIMITING REAGENT:
1. Convert mass → mol
2. Divide mol by stoich coefficient
3. Smallest value = limiting
4. Use limiting to calc product
5. Convert mol → g if needed

YIELD:
- Theoretical = max from limiting reagent
- Actual = what you got
- % yield = actual/theoretical × 100 (always ≤100%)

OXIDATION NUMBERS (priority order):
1. Free element = 0
2. Monoatomic ion = charge
3. H = +1 (except metal hydrides: H = -1)
4. O = -2 (except peroxides: -1; OF2: +2)
5. F = always -1
6. Sum = 0 neutral; = charge for ion

EQUIVALENTS & NORMALITY:
- Equivalent = amount providing 1 mol reactive units (H+, e-, charge)
- n = reactive units per molecule
- GEW = MW/n (grams per equivalent)
- equivalents = mass/GEW = mol × n
- Normality N = M × n
- Equivalence point: N_a·V_a = N_b·V_b
- Use N for polyprotic acids, multi-e- redox, salts of different charge
- HCl: n=1, GEW=36.5; H2SO4: n=2, GEW=49; Al→Al3+: n=3, GEW=9; CaCl2: n=2, GEW=55.5
