# CP Gen Chem Chapter 7 — Thermochemistry

AAMC FC mapping: FC5E (thermo) + FC1D (bioenergetics)

Kaplan: General Chemistry Ch 7

STATE FUNCTIONS (path-independent): H, S, G, U, T, P, V
PATH FUNCTIONS: q (heat), w (work)

LAWS OF THERMODYNAMICS:
- Zeroth: thermal equilibrium is transitive
- First: ΔU = q - w (AAMC convention; q+w in some textbooks)
  - Isolated: ΔU = 0
  - Adiabatic (q=0): ΔU = -w
  - Constant V (w=0): ΔU = q_v
- Second: ΔS_universe ≥ 0 for spontaneous; system can decrease if surroundings increase more
- Third: S of perfect crystal at 0 K = 0

ENTHALPY (H):
- ΔH = q_p (constant pressure)
- Exo: ΔH < 0 (releases heat); Endo: ΔH > 0 (absorbs)
- Hess's law: ΔH path-independent; reverse → flip sign; multiply → multiply ΔH; add → add ΔH
- BDE: ΔH = bonds broken - bonds formed
- ΔH_f (standard): elements in standard state = 0; ΔH_rxn = ΣΔH_f(prod) - ΣΔH_f(react)

ENTROPY (S):
- # microstates; disorder
- +ΔS: solid→liq→gas; ↑moles gas; dissolution; ↑T
- -ΔS: gas→liq→solid; ↓moles gas; combining molecules
- Phase transition T: ΔG = 0, ΔS = ΔH/T

GIBBS FREE ENERGY (G):
- ΔG = ΔH - T·ΔS
- ΔG < 0: spontaneous; > 0: non-spontaneous; = 0: equilibrium
- Spontaneous ≠ fast (kinetics separate)
- Four sign combos:
  - −ΔH + +ΔS: spontaneous at ALL T
  - −ΔH + −ΔS: spontaneous LOW T (enthalpy-driven)
  - +ΔH + +ΔS: spontaneous HIGH T (entropy-driven)
  - +ΔH + −ΔS: NEVER spontaneous
- Crossover: T = ΔH/ΔS
- ΔG° = -RT·ln(Keq); ΔG = ΔG° + RT·ln(Q)

CALORIMETRY:
- q = mcΔT
- Water: c = 4.184 J/(g·K) = 1 cal/(g·K)
- Coffee cup (const P): q_p = ΔH; for solutions
- Bomb (const V): q_v = ΔU; for combustion
- ΔH = ΔU + Δ(n_gas)·RT (significant only when gases change moles)

PHASE DIAGRAMS:
- y = P, x = T
- Triple point: solid+liq+gas coexist
- Critical point: above this, no liq/gas distinction (supercritical)
- Normal BP/MP: where line crosses P = 1 atm
- Water anomaly: solid-liq line slopes LEFT (ice less dense than water)
- Heating curves: flat regions = phase change; ΔH_vap > ΔH_fus
- Clausius-Clapeyron: ln(P2/P1) = -ΔH_vap/R · (1/T2 - 1/T1)

BIOENERGETICS:

ATP:
- Structure: adenine + ribose + 3 phosphates
- Hydrolysis: ATP + H2O → ADP + Pi, ΔG ≈ -30.5 kJ/mol
- Why exergonic: charge repulsion relief, resonance stabilization, hydration, entropy
- Intermediate transfer potential: PEP > 1,3-BPG > ATP > G6P
- Coupled reactions: add ΔG values; ATP drives anabolism

ELECTRON CARRIERS:
- NAD+ → NADH: oxidized→reduced; from niacin (B3); ~2.5 ATP via ETC
- FAD → FADH2: from riboflavin (B2); ~1.5 ATP (enters at Complex II)
- NADPH: anabolic biosynthesis (FA, cholesterol, nucleotide); generated by PPP

CHEMIOSMOTIC THEORY (Mitchell):
- ETC pumps H+ from matrix → IMS, creating proton-motive force
- Components: chemical (conc) + electrical (charge) gradients
- ATP synthase (Complex V): protons flow down gradient, drive ADP+Pi → ATP
- Inner membrane MUST be intact
- Uncouplers (DNP, thermogenin/UCP1): dissipate gradient, ETC runs, no ATP, heat
- Oligomycin: blocks ATP synthase
- Cyanide/CO: block Complex IV
- ~30-32 ATP per glucose aerobically; ~4 H+ per ATP
