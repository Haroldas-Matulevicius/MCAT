# CP Gen Chem Chapter 11 — Redox Reactions

Scope: Oxidation/reduction definitions; oxidation numbers; identifying redox; balancing half-reactions (acidic & basic); net ionic equations; disproportionation; common oxidizing/reducing agents.

AAMC FC mapping: FC4C (redox portion)

Kaplan: General Chemistry Ch 11

---

DEFINITIONS:
- Oxidation: loss of e- (oxidation state ↑)
- Reduction: gain of e- (oxidation state ↓)
- Oxidizing agent: gets reduced (causes oxidation)
- Reducing agent: gets oxidized (causes reduction)
- Mnemonics: OIL RIG (Oxidation Is Loss, Reduction Is Gain); LEO the lion says GER

OXIDATION NUMBERS (priority):
1. Free element = 0
2. Monoatomic ion = charge
3. H = +1 (except metal hydrides: -1)
4. O = -2 (except peroxide -1; OF2 +2)
5. F = always -1
6. Sum = 0 (neutral) or charge (ion)
- ID redox: any atom's oxidation # changes

COMMON OXIDIZING AGENTS:
- MnO4- (acidic) → Mn2+; (basic) → MnO2
- Cr2O7^2- → Cr3+
- H2O2 → H2O
- O2 → H2O
- Halogens (F2, Cl2) → halide
- Conc HNO3 → NO2/NO

COMMON REDUCING AGENTS:
- Active metals (Na, K, Mg, Zn)
- H2
- LiAlH4, NaBH4 (orgo: reduce carbonyls to alcohols)
- Sn2+ → Sn4+
- NADH, FADH2 (biological)

HALF-REACTION BALANCING (ACIDIC):
1. Assign oxidation numbers, ID ox + red species
2. Write two half-reactions
3. Balance atoms (not O, H)
4. Balance O with H2O
5. Balance H with H+
6. Balance charge with e-
7. Multiply for equal e-
8. Add and cancel duplicates

WORKED (acidic): MnO4- + Fe2+ → Mn2+ + Fe3+
- Reduction: 5e- + 8H+ + MnO4- → Mn2+ + 4H2O
- Oxidation: Fe2+ → Fe3+ + e- (×5)
- Sum: 5Fe2+ + 8H+ + MnO4- → 5Fe3+ + Mn2+ + 4H2O

BASIC SOLUTION:
- Balance as acidic, then add OH- equal to # H+, convert H+/OH- → H2O, cancel duplicate H2O

NET IONIC EQUATIONS:
1. Molecular equation
2. Complete ionic (split soluble strong electrolytes into ions)
3. Cancel spectator ions
4. Net ionic = remaining
- Example: Ag+ + Cl- → AgCl(s)
- Strong acid + strong base neutralization: H+ + OH- → H2O

DISPROPORTIONATION:
- Same element BOTH oxidized AND reduced
- 2 H2O2 → 2 H2O + O2 (O: -1 → -2 and -1 → 0)
- 3 ClO- → 2 Cl- + ClO3-
- Hg2^2+ → Hg + Hg2+

MCAT PATTERNS:
- Combustion: ALWAYS redox
- Single displacement: ALWAYS redox
- Metabolism: C oxidized to CO2; e- carried by NADH/FADH2 to ETC
- Analytical: KMnO4, K2Cr2O7 titrations; normality cleaner (N = M · n_e-)
