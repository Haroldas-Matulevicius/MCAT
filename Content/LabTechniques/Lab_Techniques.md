# LAB TECHNIQUES MASTER LIST

These techniques appear in passage-based questions across BB and CP. Students consistently underestimate their frequency.

---

## Separation and Purification

| Technique | Principle | When to Use |
|-----------|-----------|-------------|
| SDS-PAGE | Separates proteins by molecular weight (SDS denatures + uniform negative charge) | Determine protein size, purity |
| Native PAGE | Separates by charge, size, and shape (no denaturation) | Preserve protein activity/structure |
| Isoelectric focusing | Separates by isoelectric point (pI) | Determine pI |
| 2D gel | IEF (1st) + SDS-PAGE (2nd) | Resolve complex protein mixtures |
| Agarose gel electrophoresis | Separates DNA/RNA by size (smaller = farther) | DNA fragment analysis |
| Size-exclusion chromatography | Larger molecules elute FIRST (don't enter pores) | Separate by size, desalting |
| Ion-exchange chromatography | Binds by charge; elute by changing pH or salt | Separate by charge |
| Affinity chromatography | Binds specific ligand; elute by competitor or pH | Highest purity, specific target |
| TLC | Compound moves up plate; Rf = distance compound/distance solvent | Quick reaction monitoring |
| Column chromatography | Gravity-driven separation through stationary phase | Bulk purification |
| HPLC | High-pressure column chromatography | High-resolution separation |
| Gas chromatography | Volatile compounds separated in gas phase | Small volatile molecules |
| Distillation | Separates by boiling point differences | Purify liquids |
| Extraction | Separates by solubility in different solvents | Separate organic/aqueous |
| Centrifugation | Separates by density/size; differential = sequential spins | Separate cell components |
| Dialysis | Semipermeable membrane separates by molecular size | Remove small molecules, buffer exchange |
| Recrystallization | Dissolve hot, crystallize cold | Purify solids |

## Molecular Biology Techniques

| Technique | What It Does | Key Details |
|-----------|-------------|-------------|
| PCR | Amplifies specific DNA sequence | 3 steps per cycle: denature (95C), anneal (55-65C), extend (72C). Taq polymerase. Exponential: 2^n copies after n cycles |
| RT-PCR | Detects RNA via cDNA | Reverse transcriptase -> cDNA, then PCR |
| qPCR | Quantifies DNA in real-time | Fluorescent reporters, threshold cycle (Ct) |
| Southern blot | Detects specific DNA sequence | Gel -> membrane -> DNA probe |
| Northern blot | Detects specific RNA | Gel -> membrane -> RNA probe |
| Western blot | Detects specific protein | SDS-PAGE -> membrane -> antibody detection |
| ELISA | Quantifies antigen or antibody | Enzyme-labeled detection. Direct, indirect, sandwich formats |
| Sanger sequencing | Determines DNA sequence | ddNTPs cause chain termination at each base |
| Restriction enzymes | Cut DNA at specific palindromic sequences | Sticky ends (overhangs) vs blunt ends |
| Gene cloning | Insert gene into vector for amplification | Restriction enzymes + ligase + vector + transformation |
| Microarrays | Genome-wide gene expression | Thousands of probes, fluorescent hybridization |
| CRISPR-Cas9 | Gene editing | Guide RNA directs Cas9 nuclease to target sequence |
| Flow cytometry/FACS | Sort cells by fluorescence | Single-cell analysis, surface marker identification |
| Immunoprecipitation | Pull down specific protein + binding partners | Antibody-bead conjugate |
| ChIP | Identify DNA-protein interactions | Crosslink -> fragment -> immunoprecipitate -> sequence |
| Edman degradation | Sequence protein N-terminus | Sequential amino acid removal and identification |
| Bradford assay | Quantify total protein | Coomassie dye binding, measure absorbance |
| Karyotyping | Visualize chromosomes | Identify chromosomal abnormalities |
| FISH | Locate specific DNA sequence on chromosome | Fluorescent probe hybridization |
| Radioactive labeling | Track molecules through pathways | 3H, 14C, 32P, 35S + autoradiography |
| X-ray crystallography | Determine 3D protein structure | Requires crystallized protein |

## Spectroscopy Summary

| Method | What It Reveals | Key Details |
|--------|----------------|-------------|
| IR | Functional groups present | O-H broad, C=O sharp ~1700, N-H ~3300 |
| 1H NMR | Hydrogen environments | Chemical shift, splitting (n+1), integration |
| 13C NMR | Carbon environments | No splitting, # unique carbons |
| Mass Spec | Molecular weight, fragments | M+ peak, common losses (15, 18, 28, 29) |
| UV-Vis | Conjugation, concentration | Beer-Lambert: A=epsilon*l*c |
