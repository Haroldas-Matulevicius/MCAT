IUPAC NAMING (procedure):
1. Longest continuous chain w/ highest-priority FG = parent
2. Number to give highest-priority FG lowest locant
3. ID substituents
4. Assemble: prefix (alphabetical) + parent + suffix (FG)

CHAIN ROOTS: meth(1), eth(2), prop(3), but(4), pent(5), hex(6), hept(7), oct(8), non(9), dec(10)

PRIORITY (highest → lowest, for suffix):
- Carboxylic acid (-oic acid)
- Ester (-oate)
- Amide (-amide)
- Nitrile (-nitrile)
- Aldehyde (-al)
- Ketone (-one)
- Alcohol (-ol)
- Amine (-amine)
- Alkene (-ene)
- Alkyne (-yne)
- Alkane (-ane)
- Halide (no suffix; halo- prefix)

HYDROCARBONS:
- Alkane (-ane): saturated, CnH2n+2
- Alkene (-ene): C=C, CnH2n; cis/trans or E/Z
- Alkyne (-yne): C≡C, CnH2n-2
- Aromatic: benzene parent; ortho (1,2), meta (1,3), para (1,4); toluene, phenol, aniline, benzoic acid

ALCOHOLS (-ol):
- 1° (-OH on C with 1 C neighbor)
- 2° (-OH on C with 2 C neighbors)
- 3° (-OH on C with 3 C neighbors)
- Diols: ethylene glycol, glycerol
- Phenol: -OH on aromatic; more acidic (pKa ~10) than alkyl alcohols (pKa ~17) due to resonance

ALDEHYDES (-al):
- C=O with H on carbonyl C (terminal)
- Carbonyl C = C1 always
- Methanal (formaldehyde), ethanal (acetaldehyde), benzaldehyde

KETONES (-one):
- C=O with two R groups (internal)
- Number for lowest locant
- Propanone (acetone), 2-butanone, cyclohexanone

CARBOXYLIC ACIDS (-oic acid):
- R-COOH; carbonyl C = C1
- Methanoic (formic), ethanoic (acetic), propanoic
- Diacids: malonic (3C), succinic (4C), glutaric (5C), adipic (6C)

CARB ACID DERIVATIVES:
- Esters (-oate): R-COO-R'; "alkyl alkanoate" (methyl acetate)
- Amides (-amide): R-CO-N; peptide bond is an amide
- Anhydrides (-oic anhydride): R-CO-O-CO-R'
- Acyl halides (-oyl halide): R-CO-X; most reactive derivative
- Nitriles (-nitrile): R-C≡N; CN carbon = C1

AMINES:
- Primary R-NH2, secondary R2NH, tertiary R3N
- Methylamine, dimethylamine; aniline (aromatic); pyridine (heterocycle)

ETHERS:
- R-O-R' ; methoxymethane (dimethyl ether), THF, dioxane, epoxide (3-ring O)

COMMON NAMES TO KNOW:
- Formaldehyde, acetaldehyde, acetone
- Formic acid, acetic acid
- Methyl acetate, acetic anhydride, acetyl chloride, acetamide, acetonitrile
- Aniline, toluene, phenol, benzoic acid, pyridine
- Ethylene glycol, glycerol
