# CP Orgo Chapter 9 — Carboxylic Acid Derivatives

Scope: Acyl halides, anhydrides, esters, amides; reactivity ranking; nucleophilic acyl substitution; Fischer esterification; saponification; amide hydrolysis; reduction; lactones/lactams.

AAMC FC mapping: FC5D (acyl substitution)

Kaplan: Organic Chemistry Ch 9

---

DERIVATIVES:
- Acyl halide R-CO-X (X = Cl, Br): "-oyl halide"; acetyl chloride
- Anhydride R-CO-O-CO-R: "-oic anhydride"; acetic anhydride (Ac2O)
- Ester R-CO-O-R': "alkyl alkanoate"; methyl acetate
- Amide R-CO-NR'2: "-amide"; acetamide

REACTIVITY RANKING:
Acyl halide > Anhydride > Ester > Amide
(most reactive → least reactive)

WHY (LG quality + π-donation):
- Acyl halide: X- great LG, halogen poor π-donor
- Anhydride: RCOO- good LG (resonance-stabilized)
- Ester: RO- moderate LG, O lone pair partial π-donor
- Amide: NH2-/RNH- terrible LG, N strong π-donor → planar C-N (peptide bond), least electrophilic

NUCLEOPHILIC ACYL SUBSTITUTION (mechanism):
1. Nuc attacks C (sp²→sp³, tetrahedral)
2. LG leaves (sp³→sp², regenerate C=O)
3. Optional protonation/deprotonation
- Difference from aldehyde/ketone addition: LG present, so tet intermediate collapses

CAN GO DOWN LADDER, NOT UP:
- Acyl halide → anhydride/ester/amide ✓
- Anhydride → ester/amide ✓
- Ester → amide ✓
- Amide → harder (most stable)
- Biology: aminoacyl-tRNA = activated ester drives peptide bond (amide)

KEY REACTIONS:

FISCHER ESTERIFICATION:
- R-COOH + R'-OH + H+ + heat ↔ R-COOR' + H2O
- Reversible! Remove H2O or use excess alcohol to drive forward
- Acid-catalyzed (protonate C=O → activate)
- Reverse = acid-catalyzed ester hydrolysis (excess water)

SAPONIFICATION (base hydrolysis):
- Ester + NaOH → carboxylate + alcohol
- IRREVERSIBLE (carboxylate stable, no back-reaction)
- Soap making: triglyceride + NaOH → glycerol + Na fatty acid salts

TRANSESTERIFICATION:
- R-COOR' + R''-OH → R-COOR'' + R'-OH
- Acid or base catalyst
- Biology: phosphatidic acid → phospholipid headgroups

AMIDE HYDROLYSIS:
- Acidic: R-CONHR' + H3O+ + heat → R-COOH + R'NH3+
- Basic: R-CONHR' + OH- + heat → R-COO- + R'NH2
- HARSH conditions (heat, hours-days)
- Peptide bonds kinetically stable (~100s yrs half-life w/o protease)

REDUCTION:
- LAH:
  - Acyl halide, anhydride, ester, carb acid, nitrile → 1° alcohol or amine
  - AMIDE → AMINE (special! C=O → CH2, N stays)
- NaBH4: only aldehyde/ketone (none of these derivatives)
- DIBAL-H low T: ester → aldehyde

ACYL HALIDE + NUC (fast, irreversible, need base for HCl):
- + amine → amide
- + alcohol → ester
- Standard way to make clean esters/amides

ANHYDRIDE AS ACETYLATING AGENT:
- Ac2O + ROH → ester
- Ac2O + RNH2 → amide
- Aspirin: salicylic acid + Ac2O → acetylsalicylic acid

AMIDES NOT BASIC:
- N lone pair delocalized into C=O
- Conjugate acid pKa ~ -1 (weak base)
- Vs amine pKa ~10 (basic)

LACTAMS (cyclic amides):
- 5-mem (γ), 6-mem (δ) stable
- β-lactam (4-mem) STRAINED → penicillin core, irreversibly acylates transpeptidase

LACTONES (cyclic esters):
- 5-, 6-mem stable
- β-lactone strained
