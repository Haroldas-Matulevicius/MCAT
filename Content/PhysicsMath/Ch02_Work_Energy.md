# CP — Physics Ch 2: Work, Energy & Simple Machines

Kaplan Physics Ch 2 | FC4A

---

WORK:
- W = Fd cosθ (θ = angle between force and displacement)
  - Positive (θ < 90°): force in direction of motion
  - Negative (θ > 90°): force opposes motion (friction: θ = 180°)
  - Zero (θ = 90°): normal force, centripetal force in circular motion
- Work done by gravity: W_grav = mgh (positive when object drops)
- Gravity is conservative: work depends only on height, not path

WORK-ENERGY THEOREM:
- W_net = ΔKE (net work = change in kinetic energy)
- Bypasses kinematics for force/speed/distance problems

KINETIC ENERGY:
- KE = ½mv²
- Doubles speed → 4× KE (v squared dependence)
- Always positive

POTENTIAL ENERGY:
- Gravitational: PE = mgh (reference point is arbitrary; pick lowest = 0)
- Elastic (spring): PE = ½kx² (k = spring constant N/m; x = displacement from equilibrium)
- Only conservative forces have associated PE

CONSERVATION OF ENERGY:
- No non-conservative forces: KE_i + PE_i = KE_f + PE_f
- With friction: KE_i + PE_i + W_friction = KE_f + PE_f (W_friction is always negative)
- Key result: v = √(2gh) (object dropped from height h, or bottom of frictionless ramp)
- Shape of frictionless ramp doesn't matter — only height difference

POWER:
- P = W/t = Fv (W = watts = J/s)
- Motor lifting at constant v: F = mg, so P = mgv

SIMPLE MACHINES:
- Work in = Work out (ideal): F_in · d_in = F_out · d_out
- Mechanical advantage: MA = F_out/F_in = d_in/d_out
- Pulleys: MA = number of rope segments supporting the load
  - Fixed pulley: MA = 1 (changes direction only)
  - Moveable pulley: MA = 2
- Levers:
  - Class 1 (fulcrum between effort and load, seesaw): MA variable
  - Class 2 (load between fulcrum and effort, wheelbarrow): MA > 1
  - Class 3 (effort between fulcrum and load, tweezers): MA < 1
  - MA = d_effort/d_load
- Inclined plane: MA = L/h = 1/sinθ

KEY TRAPS:
- KE quadruples when speed doubles (v², not v)
- Friction always removes energy (W_friction always negative)
- Mechanical advantage trades force for distance — total work is unchanged
- Centripetal force does zero work (always perpendicular to velocity)
