EM SPECTRUM (low E → high E):
- Radio → Microwave → IR → Visible (ROYGBIV) → UV → X-ray → Gamma
- Visible: Red (~700 nm, lowest E) → Violet (~400 nm, highest E)
- E = hf = hc/λ | h = 6.63×10⁻³⁴ J·s | c = 3×10⁸ m/s
- All EM waves: speed c in vacuum, transverse, can be polarized, no medium needed
- Applications: MRI = radio (non-ionizing) | X-ray, gamma = ionizing (DNA damage)

REFLECTION:
- Angle of incidence = angle of reflection (both measured from NORMAL, not surface)
- Specular (smooth surface): clear image | Diffuse (rough): scattered, no image

REFRACTION:
- n = c/v (index of refraction; always ≥ 1; higher n → slower light)
- Snell's law: n₁ sinθ₁ = n₂ sinθ₂
- Into denser (↑n): slows → bends TOWARD normal
- Into less dense (↓n): speeds up → bends AWAY from normal
- Dispersion: violet bends more than red (higher n); prisms → rainbow; chromatic aberration
- Total internal reflection (TIR):
  - Requires high n → low n AND θ_i > critical angle
  - sin θ_c = n₂/n₁ (n₁ > n₂)
  - Applications: fiber optics, diamonds, viewing from underwater

MIRRORS AND LENSES (HIGH YIELD):
- Two equations: 1/f = 1/d_o + 1/d_i | m = −d_i/d_o = h_i/h_o
- Sign conventions:
  - d_o always positive (real object in front)
  - d_i positive = real image; d_i negative = virtual image
  - Mirrors: f + for concave, f − for convex; real image IN FRONT of mirror
  - Lenses: f + for converging (convex), f − for diverging (concave); real image on FAR side
  - m positive = upright | m negative = inverted | |m|>1 enlarged | |m|<1 reduced
- Diverging devices (convex mirror, concave lens): ALWAYS virtual, upright, reduced
- Converging device image behavior:
  - d_o > 2f: real, inverted, reduced
  - d_o = 2f: real, inverted, same size
  - f < d_o < 2f: real, inverted, enlarged
  - d_o = f: no image (parallel rays)
  - d_o < f: virtual, upright, enlarged (magnifying glass)
- Lens power: P = 1/f (diopters; + for converging, − for diverging)
  - Myopia: image in front of retina → diverging lens (−)
  - Hyperopia: image behind retina → converging lens (+)

WAVE OPTICS:
- Diffraction: wave bends around obstacles; significant when slit width ≈ λ
- Young's double-slit: d sinθ = mλ (bright); path diff = mλ → constructive
  - Wider λ → fringes spread | wider d → fringes closer
- Single-slit dark fringes: a sinθ = mλ (m ≠ 0); central max twice as wide
- Thin-film interference: depends on path difference (2nt) and phase shifts
  - Phase shift occurs when light reflects from higher-n surface
  - Both or neither shift (even): constructive when 2nt = mλ
  - One shift (odd): constructive when 2nt = (m + ½)λ

POLARIZATION:
- Only transverse waves can be polarized (sound CANNOT be polarized)
- Malus's law: I = I₀ cos²θ (θ = angle between polarization and polarizer axis)
  - θ = 0: I = I₀ | θ = 90°: I = 0 (crossed polarizers)
  - Unpolarized through first polarizer: I = I₀/2
  - Then through second at angle θ: I = (I₀/2) cos²θ

KEY TRAPS:
- Angles measured from NORMAL, not from surface (most common optics trap)
- Diverging devices: ALWAYS virtual, upright, reduced. Period.
- Sign conventions different for mirrors vs lenses (d_i > 0 means different things)
- Thin-film: count phase shifts carefully; constructive condition depends on whether count is even or odd
- Sound faster in denser media; light SLOWER in denser media (opposite!)
