# PS Chapter 3 — Learning and Memory

Scope: Non-associative, classical, and operant conditioning; observational learning; encoding, storage, and retrieval of memory; forgetting; memory disorders; neurobiology of learning and memory.

AAMC FC mapping: FC7A (learning), FC7B (memory)

Kaplan: Behavioral Sciences Ch 3

---

LEARNING:
- Habituation (decreased response to repeated stimulus)
- Dishabituation (recovery with new stimulus)
- Sensitization (increased response after intense event)

CLASSICAL CONDITIONING (Pavlov):
- UCS → UCR (unlearned)
- CS → CR (learned through pairing)
- Acquisition (CS before UCS)
- Extinction (CS without UCS)
- Spontaneous recovery
- Generalization, discrimination
- Higher-order conditioning
- Garcia effect (taste aversion, biological preparedness, one-trial learning)

OPERANT CONDITIONING (Skinner):
- Pos reinforcement (+ stim, ↑ behavior)
- Neg reinforcement (- stim, ↑ behavior)
- Pos punishment (+ stim, ↓ behavior)
- Neg punishment (- stim, ↓ behavior)
- Schedules:
  - Fixed-ratio: high rate, post-reinforcement pause
  - Variable-ratio: highest rate, most resistant to extinction (gambling)
  - Fixed-interval: scallop pattern
  - Variable-interval: slow steady
- Shaping (successive approximations)
- Escape vs avoidance learning
- Instinctive drift

OBSERVATIONAL LEARNING (Bandura):
- Bobo doll experiment
- Vicarious reinforcement / punishment
- Mirror neurons

MEMORY — ENCODING:
- Automatic vs effortful
- Levels of processing (shallow → deep)
- Elaborative rehearsal > maintenance rehearsal
- Self-reference effect
- Spacing effect (why Anki works)

MEMORY — STORAGE:
- Sensory: iconic ~0.5s, echoic ~3-4s
- STM/working: 7±2 (Miller), 30s, chunking
- Baddeley's working memory: phonological loop, visuospatial sketchpad, central executive, episodic buffer
- LTM:
  - Explicit (declarative): episodic + semantic — requires hippocampus
  - Implicit (non-declarative): procedural, priming, classical conditioning — does NOT require hippocampus

MEMORY — RETRIEVAL:
- Recall vs recognition vs relearning
- Retrieval cues
- Context-dependent, state-dependent, mood-congruent
- Primacy (LTM) vs recency (STM)

FORGETTING:
- Encoding failure
- Decay (Ebbinghaus forgetting curve)
- Proactive interference (old blocks new — "ProOld")
- Retroactive interference (new blocks old — "RetroNew")
- Source monitoring error
- Misinformation effect (Loftus, "smashed" vs "hit")
- Repressed/false memories

MEMORY DISORDERS:
- Alzheimer's: amyloid plaques, tau tangles, ACh deficit; cholinesterase inhibitors
- Korsakoff's: thiamine (B1) deficiency, alcoholism, confabulation

NEUROBIOLOGY OF LEARNING/MEMORY:
- Long-term potentiation (LTP) — NMDA receptors, glutamate
- Synaptic plasticity
- Hippocampus (consolidation, HM case)
- Amygdala (emotional memory)
- Cerebellum (procedural)
- Neural plasticity
