# Cell Biology Mnemonics

## Cytoskeleton Size Order: "MIcrofilaments are the MIniest"
Size goes up: Microfilaments (7nm) < Intermediate filaments (10nm) < Microtubules (25nm)
- **Microfilaments** = actin (thinnest)
- **Intermediate filaments** = keratin/vimentin/desmin/GFAP/lamins (medium)
- **Microtubules** = alpha/beta tubulin (largest, hollow tubes)

---

## Cytoskeleton Functions: "MF Move Muscles; IF Increase Integrity; MT Move Materials"
- **Microfilaments (actin):** Cell movement, muscle contraction (with myosin), cytokinesis (cleavage furrow)
- **Intermediate filaments:** Structural support/mechanical strength ONLY -- do NOT participate in movement; resist tension
- **Microtubules (tubulin):** Intracellular transport (kinesin/dynein), mitotic spindle, cilia/flagella (9+2)

---

## Motor Protein Direction: "Kinesin = Kind = Positive"
- **Kinesin:** moves toward (+) end of microtubules (anterograde, center -> periphery)
- **Dynein:** moves toward (-) end of microtubules (retrograde, periphery -> center)
- Also: "Kin-away" -- kinesin walks away from the nucleus

---

## Intermediate Filament Types: "Keep Very Decent Grades, Now Learn"
- **K**eratin = epithelial cells
- **V**imentin = mesenchymal cells (connective tissue, fibroblasts)
- **D**esmin = muscle cells
- **G**FAP = glial cells (astrocytes)
- **N**eurofilaments = neurons
- **L**amins = nuclear lamina (all cells)

---

## Cell Junction Proteins
- **T**ight junctions = **C**laudins + **O**ccludins (seal the space)
- **A**dherens + **D**esmosomes = **C**adherins
- **G**ap junctions = **C**onnexins (6 connexins = 1 connexon; 2 connexons = 1 channel)

Cytoskeletal attachments:
- **A**dherens junctions anchor **A**ctin
- **D**esmosomes anchor interme**D**iate filaments

---

## Cell Junction Order (Apical to Basal): "Tight on Top, Adherens After, Desmosomes Down, Gap Go anywhere"
1. Tight junctions (most apical -- seal)
2. Adherens junctions (belt of actin)
3. Desmosomes (spot welds, anchor IFs)
4. Gap junctions (anywhere -- communication channels)

---

## Endocytosis Types
- **Phago**cytosis = cell **eating** (large solid particles: bacteria, dead cells)
- **Pino**cytosis = cell **drinking** (non-selective fluid uptake)
- **Receptor-mediated** = **selective** uptake (e.g., LDL via clathrin-coated pits)
- Greek roots: phagein = to eat; pinein = to drink

---

## Secretory Pathway: "Really Good Proteins"
- **R**ough ER -> **G**olgi -> **P**lasma membrane
- Rough ER: protein synthesis + initial folding + N-linked glycosylation
- Golgi: modification (O-linked glycosylation) + sorting + packaging
- Also: Smooth ER = lipid synthesis + detox + Ca2+ storage
- Lysosomes = degradation (pH ~5, "Lyse stuff")
- Peroxisomes = oxidation, H2O2 production, catalase
