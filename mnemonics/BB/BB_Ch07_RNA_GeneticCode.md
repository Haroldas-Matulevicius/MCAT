## RNA Polymerase Products: "RMT" (I, II, III)
- RNA Pol **I** = **r**RNA (most abundant RNA)
- RNA Pol **II** = **m**RNA (and some snRNAs)
- RNA Pol **III** = **t**RNA (and 5S rRNA)
- "RMT sounds like 'are empty'" -- the Roman numerals need R-M-T to fill them
- Note: Prokaryotes have a single RNA polymerase (no I/II/III distinction)

## Stop Codons: "U Are Annoying, U Are Gone, U Go Away"
- **UAA**, **UAG**, **UGA** -- all start with U
- No amino acid encoded; release factors bind instead
- Only start codon: **AUG** (methionine)

## Ribosome Sites: "APE"
- **A** = Aminoacyl site (charged tRNA enters)
- **P** = Peptidyl site (peptide bond forms, growing chain sits here)
- **E** = Exit site (empty tRNA leaves)
- "An ape swings through: Arrives, does Peptide work, Exits"

## Ribosome Sizes (Svedberg NOT Additive)
- Prokaryote: 30S + 50S = **70S** (not 80S — Svedberg is a sedimentation coefficient, not a mass)
- Eukaryote: 40S + 60S = **80S**
- Mito/chloroplast: **70S** (like bacteria → endosymbiotic theory)

## Splicing
- "**EX**ons are **EX**pressed, **IN**trons are **IN**tervening (stay **IN** the nucleus)"
- Splice signals: **GU** at 5', **AG** at 3' → **GU-AG rule**

## Operon Regulation
- **LAC** = "Lacking" = normally **OFF** (inducible). Allolactose removes repressor -> transcription ON
- **TRP** = "Tripping over product" = normally **ON** (repressible). Excess Trp activates repressor -> transcription OFF
- Rule: Catabolic pathways = inducible; Anabolic pathways = repressible

### Operon Structure: "POS"
- **P**romoter (RNA polymerase binds) -> **O**perator (repressor binds) -> **S**tructural genes (protein-coding)

### Lac max expression needs BOTH:
- Lactose PRESENT (removes repressor) AND
- Glucose ABSENT (cAMP high → CAP activator)

## Epigenetics Rule
- **Acetylation = Activation** (histone — loosens DNA)
- **Methylation = Muting** (DNA — silences)

## Peptidyl Transferase
- "It's a **RIBOZYME**" — RNA catalyzes peptide bond (not a protein enzyme)
