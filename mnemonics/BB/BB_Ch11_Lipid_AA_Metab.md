## Beta-Oxidation Steps: "OHOT"
- **O**xidation: Acyl-CoA dehydrogenase (FAD -> FADH2)
- **H**ydration: Enoyl-CoA hydratase (adds water across double bond)
- **O**xidation: 3-hydroxyacyl-CoA dehydrogenase (NAD+ -> NADH)
- **T**hiolysis: Thiolase (cleaves 2C acetyl-CoA using CoA-SH)
- Each cycle yields: 1 FADH2 + 1 NADH + 1 acetyl-CoA

## β-Ox vs FA Synthesis
| | β-Ox | FA Synthesis |
|---|---|---|
| Location | Mito matrix | Cytoplasm |
| Carrier | CoA | ACP |
| Redox | NAD+/FAD (oxidized) | NADPH (reduced) |
| Rate-limit | CPT-I (inhib by malonyl-CoA) | ACC (act by insulin/citrate) |

## Carnitine Shuttle
- CPT-I = outer mito membrane, RATE-LIMITING, inhibited by **malonyl-CoA**
- Malonyl-CoA = metabolic switch between FA synthesis and β-ox

## Fatty Acid ATP Yield (Palmitate 16:0)
- 7 cycles → 7 FADH2 + 7 NADH + 8 acetyl-CoA
- ≈ **106 ATP** (after -2 for activation) — vs ~32 from glucose (fats = energy-dense)

## Odd-Chain FAs → Glucose?
- Propionyl-CoA → methylmalonyl-CoA → **succinyl-CoA** (requires **B12**)
- Only exception: "FAs can't make glucose" — odd-chain FAs can (via succinyl-CoA → OAA)

## Ketone Bodies
- Made in **LIVER ONLY** (liver can't USE them — lacks thiophorase)
- Used by: **brain (after 3-5 days starvation), heart, muscle**
- 3 bodies: acetoacetate, β-hydroxybutyrate (most abundant), acetone (fruity breath)
- DKA in T1DM: ketones > buffering → metabolic acidosis

## Lipoproteins "Chylomicrons Very Intermediately Look High"
- **C**hylomicrons (from gut) → **V**LDL (liver TAG export) → **I**DL → **L**DL (bad) / **H**DL (good, reverse transport)

## HMG-CoA Reductase
- Rate-limiting of cholesterol synthesis
- Target of **statins**

## Transamination
- All transaminases need **PLP = B6**
- **ALT** = liver-specific
- **AST** = liver + heart + muscle

## Urea Cycle — rate-limiting = **CPS-I**
- Activated by **N-acetylglutamate**
- Cost: **3 ATP per urea** (4 high-energy phosphate bonds)
- OTC deficiency = most common (X-linked)
- Urea cycle ↔ TCA linked via **fumarate** ("Krebs bicycle")

## Glucogenic vs Ketogenic AAs
- **Purely ketogenic: LL = Leu, Lys** ("LL Locked in Lipid")
- **Both: I Phe T** — Ile, Phe, Trp, Tyr, Thr (I PhoneT)
- Everything else = purely glucogenic

## Alanine Cycle (= Cori cycle for AAs)
- Muscle → alanine → liver → pyruvate + glutamate → glucose + urea
