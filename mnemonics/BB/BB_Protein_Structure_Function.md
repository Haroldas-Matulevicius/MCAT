# Protein Structure & Function Mnemonics

## Amino Acid Classifications (Kaplan System)

### Nonpolar Nonaromatic: "GAV LIMP"
- **G**lycine, **A**lanine, **V**aline, **L**eucine, **I**soleucine, **M**ethionine, **P**roline
- "Gav limps away from water because he's hydrophobic"
- Note: Matches Kaplan's 5-group system exactly. Do NOT use "Grandma Always Visits London In May For Winston's Party" (that lumps F/W into nonpolar, which doesn't match Kaplan).

### Aromatic: "WFY = Wifey"
- **W** (Trp), **F** (Phe), **Y** (Tyr) -- "You put a RING on wifey" (aromatic = ring)
- F and W are nonpolar aromatics; Y is the polar aromatic (-OH, ionizable pKa ~10.5)

### Polar: "Santa's Team Crafts New Quilts" (STCNQ)
- **S**erine, **T**hreonine, **C**ysteine, aspara**N**ine, **Q**lutamine
- Kaplan's polar group is 5 AAs (not 6). Y is classified as aromatic, not polar.

### All Charged: "Dragons Eat Knights Riding Horses" (DEKRH)
- Negative: **D** (Asp), **E** (Glu)
- Positive: **K** (Lys), **R** (Arg), **H** (His)

### Essential Amino Acids: "PVT TIM HALL"
- **P**he, **V**al, **T**hr, **T**rp, **I**le, **M**et, **H**is, **A**rg (conditionally essential), **L**eu, **L**ys
- "PVT TIM HALL always argues, never tires" -- A = Arginine (not Aspartate), T = Thr and Trp (not Tyr)

## Tricky Single-Letter Codes
- **K** = Lysine: L was taken by Leucine, so Lysine gets K (next closest in alphabet). Also: Lye = KOH = K
- **W** = Tryptophan: "Twyptophan" (Bugs Bunny). Also: indole ring looks like a W
- **D** = Aspartate: "asparDic acid"
- **N** = Asparagine: asparagiNe contains N
- **Q** = Glutamine: Q is a rounder letter for the bigger version of asparagine
- **E** = Glutamate: glutamatE

## Protein Structure Levels
1. **Primary** = sequence (peptide bonds, covalent)
2. **Secondary** = local folding (backbone H-bonds only; alpha-helix i to i+4, beta-sheet between strands)
3. **Tertiary** = 3D shape of one chain (R-group interactions)
4. **Quaternary** = multiple subunits (same forces as tertiary)

Force ranking for tertiary structure: **Hydrophobic interactions** (dominant driving force, entropic) > Disulfide bonds (strongest individual bond, covalent) > Salt bridges > H-bonds > Van der Waals

## Signal Transduction

### G-Protein Pathways: "Gs Stimulates, Gi Inhibits, Gq Squirts Calcium"
- **Gs** = stimulatory -> adenylyl cyclase UP -> cAMP UP -> PKA
- **Gi** = inhibitory -> adenylyl cyclase DOWN -> cAMP DOWN
- **Gq** = "Q-alcium" -> PLC -> IP3 + DAG -> Ca2+ release from ER + PKC

### Cholera vs. Pertussis Toxin
- **Cholera** = Gs permanently ON (can't hydrolyze GTP) -> cAMP stays HIGH
- **Pertussis** = Gi permanently OFF (locked in GDP state) -> cAMP stays HIGH
- Both increase cAMP but by different mechanisms

---

## Motor Proteins: "Kinesin Kicks Out, Dynein Drags In"
- **Kinesin** = toward (+) end (anterograde, away from nucleus)
- **Dynein** = toward (-) end (retrograde, toward nucleus)
- Also: "Kin-away" -- kinesin walks away from the nucleus

---

## Collagen: "Gly-X-Y, Vitamin C or You Die (Scurvy)"
- Triple helix, every 3rd residue = Gly (smallest, fits inside)
- X = often Pro, Y = often Hydroxyproline
- Hydroxyproline requires vitamin C (prolyl hydroxylase cofactor)
- No vitamin C = weak collagen = scurvy
