# Cardiovascular System Mnemonics — Kaplan Biology Ch 7

Covers heart anatomy, circulatory flow, and blood typing.

## Heart Valves: "LAB RAT" + "Try Pulling My Aorta"
- **LAB RAT**: **L**eft Atrium = **B**icuspid (mitral); **R**ight Atrium = **T**ricuspid
- Blood flow order: **T**ricuspid -> **P**ulmonary -> **M**itral -> **A**ortic
- Full path: Vena cava -> RA -> Tricuspid -> RV -> Pulmonary valve -> Lungs -> Pulmonary veins -> LA -> Mitral -> LV -> Aortic valve -> Aorta

## Blood Typing: "You make antibodies against what you DON'T have"
| Type | Antigens | Antibodies |
|------|----------|------------|
| A | A | Anti-B |
| B | B | Anti-A |
| AB | A and B | None (universal recipient) |
| O | None | Anti-A and Anti-B (universal donor) |
