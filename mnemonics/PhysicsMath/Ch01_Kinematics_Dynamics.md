# Kinematics & Dynamics Mnemonics — Kaplan Physics & Math Ch 1

Covers Newton's laws, kinematics equations, and basic trigonometry.

## Trigonometry: "SOH CAH TOA"
- **S**ine = **O**pposite / **H**ypotenuse
- **C**osine = **A**djacent / **H**ypotenuse
- **T**angent = **O**pposite / **A**djacent

---

## Newton's Laws: "Lazy, Formula, Pairs"
1. **Inertia** (Lazy): object stays at rest or constant velocity unless net force acts
2. **F = ma** (Formula): the only one you need to calculate with
3. **Action-Reaction** (Pairs): every action has equal and opposite reaction

---

## Kinematic Equations: "Big Five"
Each equation is missing one of the 5 variables (d, v, v0, a, t):
1. d = 1/2(v0 + v)t [missing **a**]
2. v = v0 + at [missing **d**]
3. d = v0*t + 1/2*a*t^2 [missing **v**]
4. v^2 = v0^2 + 2*a*d [missing **t**]
5. d = v*t - 1/2*a*t^2 [missing **v0**]
- Strategy: identify which variable is missing from the problem, use that equation
