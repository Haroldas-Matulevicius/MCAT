# Learning & Memory Mnemonics — Kaplan Behavioral Sciences Ch 3

Covers operant conditioning schedules, memory interference, and extinction.

## Proactive vs. Retroactive Interference
- **Pro**active = **P**ast blocks **P**resent (old memories interfere with encoding new info)
- **Retro**active = **R**ecent blocks **R**ecall of old (new memories interfere with retrieving old info)

---

## Operant Conditioning Grid
- Positive/Negative = Add/Remove (NOT good/bad)
- Reinforcement/Punishment = Increase/Decrease behavior

| | Add (Positive) | Remove (Negative) |
|---|---|---|
| **Increase (Reinforcement)** | Add pleasant stimulus | Remove aversive stimulus |
| **Decrease (Punishment)** | Add aversive stimulus | Remove pleasant stimulus |

---

## Extinction Resistance: "VR > FR > VI > FI"
- **V**ariable **R**atio: most resistant (e.g., gambling/slot machines)
- **F**ixed **R**atio: fast responding, brief post-reinforcement pause
- **V**ariable **I**nterval: slow, steady responding
- **F**ixed **I**nterval: scallop pattern, least resistant to extinction
- Rule: Variable > Fixed; Ratio > Interval
