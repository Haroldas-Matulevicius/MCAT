# Psychological Disorders Mnemonics — Kaplan Behavioral Sciences Ch 7

Covers DSM-5 personality disorder clusters.

## Personality Disorder Clusters: "Weird, Wild, Worried"
- Cluster A = "Weird" (odd/eccentric): Paranoid, Schizoid, Schizotypal
- Cluster B = "Wild" (dramatic/erratic): Antisocial, Borderline, Histrionic, Narcissistic
- Cluster C = "Worried" (anxious/fearful): Avoidant, Dependent, OCPD
