# BB (Bio/Biochem) Research Files

Deep-dive explanations for all Bio/Biochem topics. Each file mirrors its Content counterpart 1:1 but contains full teaching material: mechanisms, reasoning, worked examples, MCAT-specific tips, and cross-topic connections.

**Structure note (2026-04-28):** Both biochemistry-side AND biology-side BB files are now restructured to Kaplan 2024-2025 chapter order. Biochem files use `BB_Ch01–Ch12` prefix; biology files use `BB_Bio_Ch01–Ch12` prefix to avoid filename collision. Legacy topic-based bio files (Cell_Biology, Prokaryotes_Viruses, Cell_Division, Nervous_Endocrine, Organ_Systems, Heredity_Evolution) have been removed — their content is preserved verbatim across the new Bio_Ch files. AAMC FC mappings are inline at the top of each chapter.

## Cross-loading Rules

- If the topic involves **lab techniques** (PCR, gel electrophoresis, blotting, centrifugation), also load `research/Lab_Techniques.md`.
- If the topic involves **research methods or statistics**, also load `research/Research_Methods.md`.
- For **metabolism + organ systems overlap** (e.g., liver function, kidney filtration of metabolites), load both the relevant metabolism chapter (Ch 9–12) and `BB_Organ_Systems.md`.
