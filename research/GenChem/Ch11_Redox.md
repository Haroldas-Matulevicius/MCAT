# CP Gen Chem Chapter 11 — Oxidation-Reduction Reactions

Scope: Definitions of oxidation/reduction; oxidation numbers; identifying redox; balancing redox half-reactions in acidic and basic solution; net ionic equations; common oxidizing/reducing agents.

**Section:** Chemical/Physical (CP)
**Book:** Kaplan General Chemistry — Chapter 11
**AAMC FC mapping:** FC4C (redox portion); FC5E (redox in metabolism crossover)
**Yield:** HIGH — redox underlies electrochemistry, metabolism, and analytical titrations.

## 1. Definitions

| Term | Definition |
|------|-----------|
| **Oxidation** | Loss of electrons (oxidation state ↑) |
| **Reduction** | Gain of electrons (oxidation state ↓) |
| **Oxidizing agent** | Species that gets reduced (causes oxidation of something else) |
| **Reducing agent** | Species that gets oxidized (causes reduction of something else) |

**Memory aid:** **OIL RIG** — Oxidation Is Loss, Reduction Is Gain (of electrons).

**LEO the lion says GER:** Lose Electrons → Oxidation; Gain Electrons → Reduction.

In every redox reaction: one species is oxidized, another is reduced. Electrons transferred = balance must work both ways.

---

## 2. Oxidation Numbers (review from Ch04)

Rules in priority order:
1. Free elements = 0 (O₂, Fe metal, Na metal)
2. Monoatomic ions = charge (Na+ = +1, Cl⁻ = -1)
3. Hydrogen = +1 (except metal hydrides where H = -1)
4. Oxygen = -2 (except peroxides H₂O₂ where -1, OF₂ where +2)
5. Fluorine = always -1
6. Sum of oxidation numbers = 0 in neutral compound, = charge in polyatomic ion

**Identifying redox reactions:** Compute oxidation numbers for all atoms on both sides. If any atom's oxidation number changed, it's redox.

## 3. Common Oxidizing & Reducing Agents

### Strong Oxidizing Agents

| Agent | Common reduced form |
|-------|--------------------|
| MnO₄⁻ (permanganate, acidic) | Mn²⁺ |
| MnO₄⁻ (basic) | MnO₂ |
| Cr₂O₇²⁻ (dichromate) | Cr³⁺ |
| H₂O₂ (peroxide) | H₂O |
| O₂ | H₂O |
| Cl₂, F₂ (halogens) | Cl⁻, F⁻ |
| HNO₃ (concentrated) | NO₂ or NO |

### Strong Reducing Agents

| Agent | Common oxidized form |
|-------|---------------------|
| Active metals (Na, K, Mg, Zn) | Their cations |
| H₂ | H+ |
| LiAlH₄, NaBH₄ (in orgo) | Reduce carbonyls to alcohols |
| Sn²⁺ → Sn⁴⁺ | |
| C (in metallurgy) | CO, CO₂ |

**Biological electron carriers:** NADH/FADH₂ (reducing agents); NAD+/FAD (oxidizing agents); O₂ (terminal e- acceptor in ETC).

## 4. Balancing Redox Reactions — Half-Reaction Method

This is the systematic approach the MCAT expects. You separate the overall reaction into oxidation and reduction half-reactions, balance each, then combine.

### Steps in Acidic Solution

1. **Assign oxidation numbers** and identify the species oxidized and reduced.
2. **Write two half-reactions** (one oxidation, one reduction).
3. **Balance atoms other than O and H** in each half-reaction.
4. **Balance O** by adding H₂O.
5. **Balance H** by adding H+.
6. **Balance charge** by adding e⁻ (electrons go on the side with more positive charge).
7. **Multiply each half-reaction** so both have the same number of e⁻.
8. **Add the two half-reactions** and cancel anything appearing on both sides.

### Worked Example (Acidic)

Balance: MnO₄⁻ + Fe²⁺ → Mn²⁺ + Fe³⁺ (acidic)

**Reduction half:** MnO₄⁻ → Mn²⁺
- Mn balanced.
- Add 4 H₂O on right to balance O: MnO₄⁻ → Mn²⁺ + 4 H₂O
- Add 8 H+ on left to balance H: 8 H+ + MnO₄⁻ → Mn²⁺ + 4 H₂O
- Charge: left = +7, right = +2. Add 5 e⁻ on left: 5 e⁻ + 8 H+ + MnO₄⁻ → Mn²⁺ + 4 H₂O ✓

**Oxidation half:** Fe²⁺ → Fe³⁺ + e⁻

**Combine:** multiply Fe half by 5 to match electrons:

5 Fe²⁺ → 5 Fe³⁺ + 5 e⁻
5 e⁻ + 8 H+ + MnO₄⁻ → Mn²⁺ + 4 H₂O

**Sum:** 5 Fe²⁺ + 8 H+ + MnO₄⁻ → 5 Fe³⁺ + Mn²⁺ + 4 H₂O ✓

### Steps in Basic Solution

Same as acidic, but at the end:
8. **Add OH⁻ to both sides** equal to the number of H+ to convert all H+ to H₂O.
9. **Cancel H₂O** that appears on both sides.

Or balance as if acidic, then convert: for each H+, add equal OH⁻ to both sides; H+ + OH⁻ → H₂O on the H+ side.

## 5. Net Ionic Equations

For aqueous reactions, spectator ions don't participate — they appear unchanged on both sides.

**Steps:**
1. Write the **molecular equation** (all reactants/products as full formulas).
2. Write the **complete ionic equation** (split soluble strong electrolytes into ions).
3. Cancel **spectator ions** (those identical on both sides).
4. What remains = **net ionic equation**.

**Example:** AgNO₃(aq) + NaCl(aq) → AgCl(s) + NaNO₃(aq)

- Complete ionic: Ag+ + NO₃⁻ + Na+ + Cl⁻ → AgCl(s) + Na+ + NO₃⁻
- Spectators: Na+, NO₃⁻
- Net ionic: **Ag+ + Cl⁻ → AgCl(s)**

Net ionic equations highlight the actual chemistry. For acid-base neutralization (strong/strong): net ionic = H+ + OH⁻ → H₂O.

---

## 6. Disproportionation

A single element is BOTH oxidized AND reduced in the same reaction.

Examples:
- 2 H₂O₂ → 2 H₂O + O₂ (O goes from -1 to -2 AND -1 to 0)
- 3 ClO⁻ → 2 Cl⁻ + ClO₃⁻
- Hg₂²⁺ → Hg + Hg²⁺

---

## 7. MCAT Application Patterns

- **Combustion** — always redox (C and H oxidized; O reduced)
- **Single displacement** — always redox (free element changes oxidation state)
- **Metabolism** — every metabolic step where carbon is oxidized to CO₂ produces electrons that NADH/FADH₂ ferry to the ETC
- **Analytical titrations** — permanganate or dichromate used to titrate Fe²⁺, oxalic acid, etc.; use normality (N = M × n_e⁻) for cleaner stoichiometry

---

## High-Yield Takeaways

1. **OIL RIG / LEO GER** — oxidation = loss; reduction = gain (of e⁻).
2. **Oxidizing agent gets reduced; reducing agent gets oxidized.**
3. **Half-reaction method:** balance atoms → O with H₂O → H with H+ → charge with e⁻ → multiply → combine.
4. **Basic solution:** balance as acidic, then add OH⁻ to neutralize H+.
5. **Net ionic equations:** strip out spectators.
6. **Disproportionation:** same element oxidized AND reduced.
7. **Common strong oxidizers:** MnO₄⁻, Cr₂O₇²⁻, H₂O₂, O₂, halogens.
