# Research Files Index

This folder contains in-depth teaching material for every MCAT topic. Files mirror the Content outlines 1:1 but contain full explanations: mechanisms, reasoning, worked examples, MCAT-specific tips, and cross-topic connections.

## Folder Structure

```
research/
  INDEX.md          <-- You are here (top-level routing)
  BB/               <-- Bio/Biochem deep-dives (9 files)
    INDEX.md         <-- BB-specific routing with trigger keywords
  CP/               <-- Chem/Phys deep-dives (12 files)
    INDEX.md         <-- CP-specific routing with trigger keywords
  PS_Psych_Soc.md   <-- Psychology & Sociology (FC 6-10)
  CARS.md           <-- CARS strategy, passage analysis, question types
  Research_Methods.md <-- Study design, statistics, bias, ethics
  Lab_Techniques.md  <-- Separation, purification, molecular biology techniques
```

## How to Route

1. **Identify the subject area** from the study topic or student question.
2. **Navigate to the right index:**
   - Bio/Biochem topics → read `research/BB/INDEX.md` for file-specific routing
   - Chem/Phys topics → read `research/CP/INDEX.md` for file-specific routing
   - Psych/Soc, CARS, or cross-cutting topics → use the table below
3. **Load the matching file(s)** based on trigger keywords in the sub-index.

## Root-Level File Routing

These files live directly in `research/` (not in BB/ or CP/ subfolders).

## Cross-loading Rules

- **Lab techniques** (`Lab_Techniques.md`) should be co-loaded whenever a BB or CP topic involves specific experimental methods.
- **Research methods** (`Research_Methods.md`) should be co-loaded whenever the topic involves study design, data interpretation, or statistical reasoning.
- Always load the **Content sub-file** (from `Content/`) alongside its matching Research file for complete coverage.
